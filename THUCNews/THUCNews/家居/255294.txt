　　墙面材料
　　墙面是室内建筑空间的重要组成部分，而且是家居空间中视觉敏感性最强的部位，墙面装修的好坏直接影响着家居装修的整体效果，而要有好的装修效果，必须准确选择材料，下面按施工方法不同简单介绍一下墙面装修的常用材料。
　　一、 涂料类材料
　　墙面材料是重要的饰面材料，它具有施工简单，工期短，自重轻，维修方便，造价低等优点。
　　目前室内常用的装修涂料有： 106 内墙涂料， 803 内墙涂料，多彩涂料，调和漆，磁漆 ( 俗称刮磁 ) ，乳胶漆等 ( 俗称喷漆 ) 。
　　二、 裱糊类材料
　　裱糊饰面是采用壁纸，锦缎等，粘贴于墙的?面，以达到装饰效果，它具有多彩丰富，图案变化多样，美观耐用，施工方便等优点。
　　用于家庭室内装修的壁纸和贴墙布主要有：塑料壁纸，织物壁纸，复合壁纸，纺织贴墙布，无纺贴墙布，玻璃纤维墙布，金属膜壁纸，电植绒壁纸，装饰壁布 ( 宜用于软包墙面 ) 等。
　　三、 贴面类材料
　　贴面类材料主要有两大类：饰面砖和饰面板。饰面板如大理石板，花岗石板等，但进价较高，一般用于外?饰面，内?饰面特别是家庭装修中很少采用。
　　常用饰面砖有瓷瓦，陶瓷锦砖 ( 马赛克 ) ，玻璃锦砖 ( 玻璃马赛克 ) 。具有独特的卫生易清洗和清新美观的装饰效果，在家庭装修中常用于厨房，卫生间等的?面。
　　四、 罩面板类材料
　　此类饰面多以轻钢龙骨，铝合金龙骨，木龙骨为骨架，以各类罩面板进行罩面，以达到装饰效果，常用于室内?面，顶棚，隔墙等处。
　　家庭装修中常用得罩面板材有：
　　木质板 如胶合板，纤维板，木丝板，印刷木纹板，微薄木贴面板。
　　塑料板 如铝塑装饰板，泡沫塑料装饰板，塑料装饰板。
　　石膏板 如纸面石膏板，纤维石膏板，浮雕石膏板，石膏装饰板。常用于墙面和顶棚。
　　纤维板 如矿梯装饰板，玻璃面装饰板，水泥装饰板。常用于室内隔?。
