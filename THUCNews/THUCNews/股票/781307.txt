格罗斯：未来一段时间美国可保住AAA信用评级
　　新浪财经讯 北京时间3月12日早间消息，太平洋投资管理公司(PIMCO)联席首席投资官、有“债券之王”之称的比尔-格罗斯(Bill Gross)周五称，未来一段时间里美国将可保住其“AAA”主权信用评级。
　　格罗斯在今天接受采访时称，与其他主权债券和部分公司债券相比，美国国债对投资者的吸引力相对较低。他指出，与正常水平相比，当前美国国债的收益率要低了150个基点左右。昨天有消息称，上个月格罗斯已经将麾下旗舰基金Total Return Fund中的政府相关债券清零，原因是预计美国政府的财政赤字将会创下历史最高水平。Total Return Fund上一次完全不持有政府相关债券是在2009年1月份。
　　据太平洋投资管理公司网站上公布的数据显示，1月份格罗斯已将Total Return Fund基金所持政府相关债券在总资产中所占比例削减至12%。在2月份中，Total Return Fund基金的净现金及现金等价物头寸从此前的5%上升至23%，创下自2008年5月份以来的最高水平。Total Return Fund基金是目前世界上最大的共同基金。
　　据美国银行美林公司(Bank of America Merrill Lynch)编纂的指数数据显示，2010年中美国国债的回报率为5.9%。在去年第四季度中，美国国债的回报率大幅下降了2.7%，但今年截至目前为止则上升了0.1%。另据彭博社编纂的数据显示，Total Return Fund基金过去一年中给投资者带来的回报率为7.43%，好于84%的同业竞争对手；过去一个月中，该基金的回报率为1.86%。
　　太平洋投资管理公司网站显示，该公司所持政府相关债券类别可包括传统国债、通胀相关国债、机构债、利率衍生品、国债期货和期权、以及由联邦存款保险公司(FIDC)支持的银行债券。衍生品是与股票、债券、贷款、货币和商品相关，或是与利率或天气变化等特殊事件相关的不受监管的金融工具，其功能是降低潜在资产的损失风险。
　　太平洋投资管理公司是安联(Allianz SE)(ALV)旗下部门。截至去年12月份为止，太平洋投资管理公司管理下的客户资产总额为1.24万亿美元。(文武)
