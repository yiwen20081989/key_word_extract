光亚否认旗下First Media有洽商及有意出售
　　新浪财经讯 1月21日消息，光亚(08061)旗下 PT First Media Tbk 否认正待出售之新闻报导，亦确认 First Media 公司方面既无曾经讨论，亦无意作出售。
　　光亚发布公告称，，旗下 PT First Media Tbk 否认最近各种正待出售之新闻报导，亦确认 First Media 公司方面既无曾经讨论，亦无意作出售。董事会在最近与 First Media 沟通中，得知其已委任美林新加坡为财务顾问进行商务策略之策略性检视及增长计划提供建议，此将提升股东之价值。
　　光亚指出，最近传媒报导 SK Telecom Co. 已提交一份对光亚母公司印度尼西亚力宝集团之有线电视和互联网公司 First Media 的投标，及力宝集团正在为 First Media 索价6亿美元至8亿美元。董事会谨此澄清，并不知悉该有关 SK Telecom Co.报导之投标，或任何First Media 据称中的出售。
