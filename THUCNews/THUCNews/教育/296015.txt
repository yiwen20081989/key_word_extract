提高自信少焦虑：考前心态调整小测试及答案 
　　迎考试卷(总分100分)
　　一、多项选择题(总计40分，每小题5分)
　　1、“烦”就一个字，却说了无数次。如何才能化解频频来袭的烦躁？( )
　　A、去酒吧“嗨”一晚，把烦恼甩光光。
　　B、蒙头大睡一觉。
　　C、跟朋友或亲近的师长倾诉，打一小会球，在操场慢跑，在阳台上做体操，暂时放下难题，换一个学科，做一点家务，听一小段舒缓的乐曲……
　　D、叫上几个哥们，今宵不醉不罢休。
　　2、几次模拟考试或调考，总觉得没准备充分。平时总担心高考考不好怎么办？( )
　　A、首先，一定要走出题海战术，着重查漏补缺；然后，目前要拒绝难题、偏题了，以免影响自信；最重要的则是要学会说“我能行！”抱着只要尽力就行的心态，则可以难易不惊。
　　B、这么大的事，担心很正常，不用去理会。
　　C、担心正好说明了对高考的重视，既然重视了就会有好结果吧。
　　D、把头埋进书堆中，多记多背多做题，别的自然就不会去想了。
　　3、高一高二荒废时间太多，基础没打牢。高三复习时感到漏洞百出，因此懊悔不迭。该如何是好？( )
　　A、早知如此，何必当初，是应该懊悔嘛。
　　B、懊悔无济于事，看看成绩更差的同学，心里就会平衡些了。
　　C、抱着“破罐子破摔”的心态反而可以从容应对，说不定可以出现“奇迹”。
　　D、悟已往之不谏，知来者之可追。对于智者而言，他的“智”不在于没有“不谏”，而在于这“不谏”是为他的“追”而存在的。不要悔，给自己当头一喝！查缺补漏，早日迎头赶上。
　　4、习惯了“开夜车”，到了白天昏昏欲睡，如此疲惫怎么办？( )
　　A、反正上课讲的都是以前的知识，睡睡无妨。
　　B、考前必须把兴奋点调到白天来，为保证在高考时达到理想状态，考前一周应尽量早睡早起，把生物钟调整到“高考步调”。
　　C、考前休息好最重要，要抓紧一切时间睡觉，这样就不会疲惫了。
　　D、考前冲刺最重要，疲惫一时、受益一生，撑着眼皮也要挺住！
　　5、总睡不踏实怎么办？()
　　A、睡不着就起来做题。
　　B、睡前去操场上狠命跑几圈，跑累了就会犯困。
　　C、如果失眠，千万不要服用药物。偶尔失眠并不可怕，实在睡不着，安静休息就行了。
　　D、在最后的复习阶段，晚饭后适当安排学习即可，不要超过11点就寝。最好不要看完书就直接上床，别在睡前做太难的题。
　　6、不知道为什么变得特迷信。常去烧香拜佛、身上带着玉佛、早上听到乌鸦叫就紧张万分。到底该不该在意这些呢？( )
　　A、信则有，不信则无。与其不信，不如信一下。
　　B、迷信没有用，不要相信这些外在的东西，而要相信自己的努力。
　　C、“喜鹊报喜，乌鸦惹祸。”中国人信了几千年，怎么可以不在意呢？
　　D、不是说“临时抱佛脚”吗？抱总比不抱好嘛。
　　7、平时还好，可一到大考就心跳加快、血压升高、面红耳赤、反常遗忘……怎么办？( )
　　A、呼吸松弛法。进考场感到慌张时，可双目微闭，深呼吸三四次，尽量做到绵长。
　　B、自我暗示法。考试中，如果意识到紧张，心中可默默对自己说：“我要冷静，要放松。”“我有信心考出最好水平。”
　　C、解除疲劳法。考试时可给自己一些调整状态的短暂间歇，如伸展四肢、活动手腕头颈、摇摇手指关节等。
　　D、集中注意法。当注意力不能集中，可不急于答题，握住笔杆，两眼盯著前面的某个固定对象，有意缩小视野，几分钟后再集中到题目上来，这样可让心神安定。
　　8、考试时，遇到熟悉的题目还能得心应手，碰到难题，就有如五雷轰顶的感觉。( )
　　A、遇到难题，务必不要焦虑，更不能乱了阵脚。可暂时搁置一边，先做会做的，保证已做出题目的正确率，然后再考虑难题。
　　B、看到不会做的就立即放弃，不要耽误时间了。
　　C、高考的胜负就在于难题，所以要“不惜一切代价”攻克。
　　D、考试就像人生，应“先苦后甜”，所以要先做难题，再做简单的。

　　简答题
　　稳定心态必不可少
　　稳定的心理状态、扎实的学科知识和灵巧的应试技能是考出好成绩的必备条件。
　　某位教授曾连续5年对高考状元进行跟踪研究，发现在影响高考成功的20个因素中，最重要的因素是考场心态，其次是考前心态，第三位是学习方法，第四位是学习基础。
　　问答题
　　1、十大秘诀 助考生轻装上阵
　　(1)、接受现状，目标期待适当。
　　考生要根据平时的学习实力和心态情况，实事求是地确定高考目标。如果目标过高，就会为难以达到而增加焦虑；如果目标太低，又会影响发挥。
　　(2)、只和自己比，不和他人比。
　　每个考生的学习实力与心态状况不一样，高考期待也不一样。有的考生盲目与别的同学攀比，这样会挫伤自信。只要在高考中发挥出平时的水平就是成功。
　　(3)、充分休息，调整竞技状态。
　　与其以丧失状态为代价而在最后阶段争分夺秒地做题，不如放松下来以激发最佳竞技状态的爆发。不过，也不要走另一个极端，将放松变成放纵。
　　(4)、积极自我暗示，强化信念。
　　积极的暗示能增强信心，消极暗示会降低信心。考前，要特别注意用积极暗示进行心理调整，强化信心。
　　(5)、挺胸抬头，走路步伐加快。
　　可利用人的内心体验和行为动作一致的原理，每天走路挺胸抬头，步伐稍快。经常这样做，可增强自信。
　　(6)、坦然面对，不怕考试焦虑。
　　考试前和考试中存在轻度的考试焦虑相当普遍。可有些同学把焦虑看得太重。其实，谁过分看重考试焦虑，谁就会受到影响。
　　(7)、正确对待，学会自我减压。
　　以怎样的态度对待考试，对进入状态关系很大。若把考试看成挑战，会激发自己进入状态；若把考试看成是机会，则会以一种珍惜的心情对待；若把考试看成如临大敌，就会紧张不安。
　　(8)、丢掉包袱，轻装上阵。
　　师长的期望、过高的目标都会添加压力。过多的压力会使考生心烦意乱，不能集中精力。其实，从某种意义上说，考试成绩考前就已决定，无需忧心忡忡。所要做的一切就是顺其自然，发挥出实际水平。
　　(9)、消除紧张，平静心情。
　　进入考场的第一件事，就是要使自己心情镇定下来，坐在座位上深呼吸几次或面对窗外，向远处眺望一会，以平静心情。
　　(10)、堤内损失堤外补，学会遗忘。
　　考一科，忘一科，一考百了，不对答案、不议论、不想考试结果，暗示自己"还行"，要越考越有信心。
　　2、五大方法 让考生调整放松
　　(1)、听音乐。尤其是听轻音乐对调整人的紧张情绪的效果是非常好的。
　　(2)、积极自我暗示。考生在高考复习过程中与在考场上，都可根据自己当时的心态进行积极的暗示。考生根据自己的情况采用不同的自我暗示语。当情绪过于紧张时可默念“放松、放松、放松”；当心灰意懒时可默念“我行、我行、我行”。(3)深呼吸。这是调整情绪的一种简便方法，闭目坐在椅子上，缓慢有节奏地吸气，停几秒钟后，再缓慢有节奏地呼气。
　　(4)、按摩内关穴。用自己右手大拇指轻轻按摩左前臂上的内关穴(位于手腕横纹处向肘部方向三横指正中线上)，按摩时要轻轻而有节奏地顺时针方向进行，按摩36次即可。(5)想象排忧。可站着进行训练，也可在晚上淋浴时训练。训练时开始想象天上下着小雨，雨水顺着面部流到胸部、腹部，同时顺后脑流到背部、腰部……一面想象雨水沿身体向下流，一面想象雨水把烦恼、忧愁、紧张等统统冲掉。
