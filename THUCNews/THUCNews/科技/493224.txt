强悍DC上市无敌兔降价 热门DC排行榜
　　在7月和8月份，通常是各大数码相机品牌发布旗下新品的大日子，各种新鲜技术、新鲜功能的出现，又将会掀起数码相机向更高的性能迈进。在过去的几周时间里，已经陆续有奥林巴斯、松下、富士、索尼等品牌推出了2010年下半年的新机。机型本周热点机型看点参考价格
　　佳能EOS 5D MarkII首先在数码单反相机方面，佳能人气全画幅机型EOS 5D MarkII的价格在本周特别受人关注，虽然不是大幅度降价，但目前的单机最新售价已经降为16280元，与同是全画幅数码单反，目前售价为14500元的尼康D700的差距正……16280元
　　松下 GF1(套机)最近，索尼首款单电相机NEX-5的人气爆棚，完全掩盖了同样是单电相机的Macro 4/3系统。但在本周，来自Micro 4/3系统联盟的松下，将旗下最具人气的单电相机GF1套机（含20mm F1.7饼干镜头）价格狠降至4700元的幅度……4700元
　　索尼 WX1在卡片数码相机方面，本周又有惊喜了。采用索尼背照式Exmor R CMOS感光元件，并搭载了24mm超广角镜头的轻巧卡片机索尼WX1报出超值价1500元，对于近期要购买相机的朋友，不妨好好的考虑索尼WX1噢，因为它的性价比太高了，而且又拥有非常时尚的外表……1500元
　　富士 Z808(新上市)虽然各品牌的2010年下半年新品才刚发布，但在本周已经有新品抢先上市的消息了。首先上市的新品有富士3.5英寸大屏幕+时尚滑盖卡片机FINEPIX Z808，从它上市定价为1560元来看，Z808依然属于高性价比卡片机的类别……1560元
　　松下FZ100(新上市)在本周的数码相机市场上，我们还迎来了一款高端长焦DC的登场，它就是松下新发布的专业长焦数码相机DMC-FZ100。松下FZ100在性能配置上得到了全面的革新，无论在性能、配置方面，均要比FZ35要强大很多。在价格上，这款旗舰长焦一点也不轻松，新品报价高达3580元……3580元
　　在本周，我们又迎来了尼康及佳能的部分新品，相信到了8月末，才真正是好戏上演的时候，毕竟佳能在本周仅发布了便携长焦IXUS 1000 HS、PowerShot SX130 IS 和专业便携机 PowerShot S95。而尼康方面，除了新升级的内置投影器数码相机S1100pj 和 S5100这两款卡片DC外，还有入门级数码单反D3100及4支不同级别的全新G镜头和我们见面。而在笔者的角度看，尼康和佳能本周发布的机型算是前凑，让人期待的高端数码单反、高端数码相机，估计还在后头吧！ 
　　好了，在简单的总结了本周数码相机新品的一些情况之后，我们再来看看本周数码相机市场方面的信息，本周的热门DC排行榜，又会有哪些机型上榜？下面我们马上揭晓答案。 
　　人气全画幅数码单反 佳能EOS 5D MarkII  
　　最新报价：16280元（单机） 
　　
　　首先在数码单反相机方面，佳能人气全画幅机型EOS 5D MarkII的价格在本周特别受人关注，虽然不是大幅度降价，但目前的单机最新售价已经降为16280元，与同是全画幅数码单反，目前售价为14500元的尼康D700的差距正在进一步缩小。不知道佳能无敌免的此番降价，是不是要扩大其在全幅数码单反方面的影响力呢，因为尼康很有可能在近期发布一款定位在D700以下的低价全幅机，以扩大尼康在中低端全幅数码单反市场上的占有率。虽然尼康要推出定位于D700以下的全画幅数码单反还仅限于网络传言的阶段，但是佳能无敌免的降价，一定会加剧中低端全幅机型的竞争。 
　　佳能EOS 5D MarkII能够如此受到关注，除了它是一款全画幅机型以，它的其它配置也十分高端，拍摄画质亦非常优秀。相机配置了一块有效像素达到2100万的全画幅（36mm x 24mm）CMOS，再配合上全新的DIGIC 4数字影像处理器，使EOS 5D MarkII拥有极为出色的影像品质。 
　　
　　EOS 5D Mark II采用9点宽区自动对焦系统，除了9个自动对焦点，另有6个辅助对焦点。而相机最高快门速度达到1/8000秒，闪光同步速度是1/200秒，标准感光度范围为ISO 100–3200，可扩展至最高的ISO 25600，当然了，5D Mark II是一款拥有极为出色控噪能力的全幅机。另外，EOS 5D Mark II还可以拍摄全高清电影，最高分辨率是1920×1080像素（16：9）。5D Mark II的屏幕为一块3.0英寸、92万点分辨率的宽视角防反射清晰LCD，并支持实时取景。 
　　
　　作为一款中端全画幅数码单反相机，EOS 5D Mark II机身使用了轻巧而坚固的镁合金材料，并具有一定的防尘防水滴性能。在如些强大的配置下，EOS 5D Mark II的售价在16000元左右，还算比较公道了。 
　　热门MICRO 4/3单电相机 松下GF1  
　　最新报价：4700元套机（含20mm F1.7镜头） 
　　
　　最近，索尼首款单电相机NEX-5的人气爆棚，完全掩盖了同样是单电相机的Macro 4/3系统。但在本周，来自Micro 4/3系统联盟的松下，将旗下最具人气的单电相机GF1套机（含20mm F1.7饼干镜头）价格狠降至4700元的幅度，与索尼的NEX-5形成正面的交锋，至于谁才能成为大赢家，现在还很难下一下结论，但小编觉得，松下GF1配这支大光圈定焦镜就更为及引了，而且在拍摄操作方面，松下GF1也要比索尼NEX-5来得清爽，因为索尼NEX-5的拍摄操作被弱化，对于专业用户来说很不习惯。 
　　
　　虽然在感光元件上，索尼NEX-5的Exmor APS HD CMOS要比GF1的Live MOS在面积及像素方面都要有优势，但在成像质素上却并未有太大的差异，而且GF1的1210万有效像素已能满足我们的拍摄所需。另外在对焦系统方面，GF1采用的是反差自动聚焦系统，所以在对焦速度上，比其它单电相机要有优势。 
　　
　　另外，GF1上还设置了专用的录影快捷键，相机支持AVCHD Lite格式的高清动画录影，帧率为60P/30P，而拍摄质量也分别有17Mbps，13Mbps，9Mbps可选。松下GF1支持机身闪光灯，也支持热靴外接闪光灯，这一点索尼NEX-5就无法超越了。此外，松下还针对GF1而设计了专用的EVF电子取景器，放大倍率为1.04X。而GF1配有一块3.0英寸46万像素LCD，显示比例为3：2。 
　　背照式CMOS卡片机 索尼WX1  
　　最新报价：1500元 
　　
　　在卡片数码相机方面，本周又有惊喜了。采用索尼背照式Exmor R CMOS感光元件，并搭载了24mm超广角镜头的轻巧卡片机索尼WX1报出超值价1500元，对于近期要购买相机的朋友，不妨好好的考虑索尼WX1噢，因为它的性价比太高了，而且又拥有非常时尚的外表。 
　　
　　索尼WX1采用的是1/2.4英寸背照射式“Exmor R”CMOS，虽然有效像素为1020万，但它的感光性能是传统CCD/CMOS的两倍之多。镜头方面则采用了索尼G级镜头，支持5倍光学变焦，等效焦距达到了24-120mm，最大光圈为F2.4-5.9，微距约5cm，并支持镜头光学防抖。而背照式CMOS的优异感光性能与F2.4超大光圈的配合，已经可以横扫各种弱光环境的拍摄了。 
　　
　　其它方面，WX1的感光度为ISO 160/200/400/800/1600/3200，相机支持“手持夜景模式”、10张/秒连拍和“扫描全景”模式，另外还支持1，280×720 @ 30fps高清视频录影。相机的屏幕为2.7英寸23万像素高清晰LCD。WX1的机身尺寸为90.5×51.8×19.8mm，仅重129g。 
　　
　　以WX1的配置，在众多的卡片数码相机当中已经是出类拔萃的了，所以它也无惧新品DC的正面挑战，而目前它的身价已经跌至1500元，是如此之高的性价比啊。 
　　新品来袭 富士Z808上市 
　　最新报价：1560元 
　　
　　虽然各品牌的2010年下半年新品才刚发布，但在本周已经有新品抢先上市的消息了。首先上市的新品有富士3.5英寸大屏幕+时尚滑盖卡片机FINEPIX Z808，从它上市定价为1560元来看，Z808依然属于高性价比卡片机的类别。 
　　
　　定位为高性价比系列的Z808采用的是富士1/2英寸的Super CCD EXR，其有效像素为1200万级别。镜头则采用了超薄卡片机常用的潜望式变焦镜头，等效焦距为35-175mm，最大光圈为F3.9-F4.7。虽然镜头没有28mm的广角端，而且最大光圈也没有什么优势，但是支持5倍光学变焦已经不错，而且相机支持光学防抖。另外，屏幕是Z808的一个亮点，因为它采用了一块16:9规格的3.5英寸46万像素TFT液晶屏，而且是全触摸式操作。 
　　
　　在拍摄功能方面，Z808除了支持P档程序自动外，还支持多种预设场景模式。在对焦速度方面，Z808采用了全新的高速自动对焦功能，它将对比检测AF和相位检测AF无缝融合，达到0.158秒的极速对焦速度。Z808对于时下非常流行的全景拍摄同样支持，而且它的拍摄角度甚到达到了360，表现极为出色。相机的视频拍摄能力达到了720P高清规格。 
　　
　　新品定价仅为1560元，只能说富士Z808的性价比很高了。 
　　新品来袭 松下高端长焦FZ100上市 
　　最新报价：3580元 
　　
　　在本周的数码相机市场上，我们还迎来了一款高端长焦DC的登场，它就是松下新发布的专业长焦数码相机DMC-FZ100。松下FZ100在性能配置上得到了全面的革新，无论在性能、配置方面，均要比FZ35要强大很多。在价格上，这款旗舰长焦一点也不轻松，新品报价高达3580元，是觉得有点儿贵了。 
　　
　　拥有如此高的售价，在配置方面松下FZ100也足够强大的了，它采用1410万像素的MOS传感器，并采用了Venus Engine FHD图像处理引擎，并提供了RAW+JPEG的图像储存格式。镜头方面，FZ100采用了24倍光学变焦，并拥有等效焦距为25mm - 600mm的LEICA DC VARIO-ELMARIT镜头，最大光圈为F2.8-F5.2，拥有24倍光学变焦能力的FZ100也可当望远镜之用了。 
　　
　　机身其它特点方面，松下FZ100配备一块3.0英寸46万色的LCD屏，支持旋转操作，所以在仰视或俯视拍摄时，这块屏幕都能提供很舒的拍摄角度。FZ100另一特点是支持机顶热靴插槽，通过热靴，可以外接闪光灯及立体声收音麦克风。 
　　
　　在拍摄功能方面，FZ100支持光圈优先、快门优先、全手控曝光等操作，而且还提供了多达23个对焦。另外，相机的感光度为ISO 80 / 100 / 200 / 400 / 800 / 1600（高感光度模式1600-6400）。连拍性能方面，即使在全像素下，也可以达到每秒连续拍摄11张，在350万像素下能以60张/秒的连拍速度进行记录。其实松下FZ100视频拍摄能力惊人，在一般Motion JPEG格式下支持1280 x 720高清录影，而在AVCHD PAL格式下，则支持全高清1920 x 1080 60i(NTSC) / 50i (PAL)的视频记录格式。 
　　
　　松下FZ100的综合性能配置强大，是一款高端专业长焦数码相机，但是面对3580元的高价位，也并不是每个人都能承受的。
