Symbian Belle系统上手初评
　　诺基亚最近保密工作不太好。上周官方刚对 N8，E7 等机型推送 Symbian Anna 更新，国外有论坛放出了下一代系统 Symbian Belle 的固件下载。事实上 Belle 的 UI 早先已经被媒体曝光，用户界面和一些过渡效果相比 Symbian^3 PR1.2 和 Symbian Anna 都有不小的变化，其中最明显的要数下拉式通知栏。现在给大家说一些上手感觉。
　　升级
　　我手里的 N8 先前是 PR 1.2 系统，OTA 升级到 PR 2.0 即 Symbian Anna。看到 Belle 固件流出后，用凤凰 Software Update 模式升级。过程很快，5 分钟左右就 OK，而且之前安装的软件以及保存的通讯录、通话记录、短信都在。就我升级体验而言，不存在外界说的需要格机操作。先给一张版本图，由于是泄露版，所以未 标具体代号。
　　用户界面改进
　　Symbian Belle 和之前系统最大的不同就是用户界面的改进：引入下拉式通知系统，修改 Widget 样式，系统界面要素调整。可以看下对比，作为Symbian^3 PR 1.2，右为 Symbian Belle。
　　这里可以看到诺基亚在 Belle 抛弃了从最早 S60 以来的界面上方左信号右电量的布局，和其他包括时间、运营商和 Wi-Fi 信息等统统移到顶栏。
　　同样，界面下方“左右软键”布局也得到调整，引入类似 S60 V3 FP2 部分机型的左、中、右按键布局。Symbian 直觉式的左选项、右退出变成了左返回、右菜单。程序列表没有变化，就是图标做了修改，设计风格从 Anna 开始改变。原本期盼的 4 x 3 或是 4 x 4 界面布局没有出现，图标整体显大，不是很协调。后期应该可以通过自定义固件方式修改
　　Symbian 触摸机型最让我费解的弹出式输入框终于取消，引入独立 qwerty 虚拟键盘(很快国外就有了改进版，数字行位于字母上方)。英语输入有单词提醒修正功能。由于是泄露版的关系，系统不含中文输入法。
　　桌面 Widget 样式也做了改进，出现了翻页时钟。之前版本中快捷方式只能四个一排连体出现，现在可以有类似 Android 和 iOS 的单个图标拖拽。另外多任务界面和 PR 1.2 相比没有太大变化。
　　字体在我看来还是不太美观。给我留下颇佳印象的 Nokia Pure 字体没有出现，早先 Symbian Foundation 公布的字体也没有。这点比较遗憾，希望正式推送 Belle 的时候能有改进。
