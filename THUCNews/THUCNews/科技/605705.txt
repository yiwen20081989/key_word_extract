打造最完美桌面 实现PPC全屏壁纸教程
　　PPC的桌面非常美观，并且能根据自己的喜好随意更换壁纸。不过自定义的壁纸并非全屏的，上下两边露出的工具栏与自定义的壁纸非常不协调。下面就给大家介绍一个解决方案，这个设置虽然很麻烦，有兴趣的并且孜孜不倦的再照着设置，辛苦能换来收获。
所需软件：（1）电脑美工编辑软件，我是用 PhotoCap，百度一下很多。（2）RescoExplorer，修改系统文件属性及复制用。
前言：　　首先看第一张图，说明 i8000 桌面的分割，最上面是 Task bar 也就是状态栏或工作栏。宽 480 高 52 像素，最下面是 Softkey bar 也就是软键菜单栏，一样是 480 x 52。正中央的才是有效壁纸区，宽 480 高 696。尽管 i8000 号称可以显示 480 x 800 的壁纸，但实际上仅仅用到蓝色中央区域的 480 x 696，其余上下栏都被保留了不能用。
　　经过搜索注册表，得知上下栏在 Windows 文件夹里根据主题分别被定义成 480 x 52 的“某种颜色”。以 Classic Blue 为例，这二栏被定义成黑色。文件名分别是 black_softkeybar.png 和 black_taskbar.png，使用 RescoExplorer 得知其文件属性为“只读”及“系统”。　　了解以上现况后，接下来我们就可以藉由修改这二个文件的内容使其显示的内容与我们的壁纸“相呼应”，如此就可以让壁纸“扩展”达到全屏的效果。　　首先找一张喜欢的壁纸，分辨率 480 x 800，从电脑端打开美工编辑软件载入，使用软件的“裁切”功能截取最上面 480 x 52 部分，储存好稍后备用（文件名可以先用 Task Bar，后缀名一定要是 PNG，格式 24位元），然后将文件复制到手机 SD 卡稍后使用。　　接着同上截取最下面 480 x 52 部分，一样储存备用，文件名可用 Menu Bar，后缀名及格式同上。一样复制到手机 SD 卡。　　接下来是最关键的部分，那就是剩下的还没有用到的壁纸部分（分辨率为 480 x 696）。使用美工编辑软件将“版面”放大为 480 x 800，图形部分要往上贴齐，下方留下空白无所谓，然后用好记的文件名储存。　　这样已经完成电脑端的准备工作，最后就剩手机部分的工作。使用 RescoExplorer 找出 Windows 文件夹下面的 black_taskbar.png 和 black_softkeybar.png，先备份到 SD 卡里以备不时之需。　　打开 RescoExplorer，把刚刚的 Task Bar.PNG 重新命名为 black_taskbar.png，Menu Bar.PNG 重新命名为 black_softkeybar.png，再以 RescoExplorer 更改其属性在【属性】中把“只读”和“系统”勾选上，其他一律不要勾选。　　最后一步就是用 RescoExplorer 把这二个文件复制到 Windows 【强制】覆盖原本的文件即可。然后在手机里选择壁纸为刚刚“被截去上下栏”且经重新处理过的壁纸，重新开机后就可以看到“三者”被巧妙的“接合”了。
　　注意事项：如果更换其他主题上下栏可能也会一并更改，此时以上步骤需重新做过，欲更换新的壁纸时亦同。
