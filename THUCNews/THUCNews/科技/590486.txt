嵌入式长尾提速：微软掘金小屏幕
　　嵌入式领域蕴藏的巨大市场空间将在后PC技术的时代爆发。对于微软来说，占领嵌入式的长尾也就意味着占领人们未来生活的方方面面。
　　本刊记者  马荟
　　比尔·盖茨在微软创立30年后向世界预言：机器人将与30年前的个人电脑一样迈入家家户户，彻底改变人类的生活方式。而机器人就将是嵌入式存在的最高形式。英特尔CEO欧德宁也曾表示“嵌入式系统将是英特尔的下一座宝藏”。
　　人们对嵌入式有诸多定义，比如“嵌入式设备就是一类想要不成为计算机的计算机”，可以看出嵌入式设备拖着的长长的尾巴，已经孕育了PC之外的巨大市场空间。
　　事实上，任何一个普通人可能拥有从大到小的各种使用嵌入式技术的电子产品，小到MP3，PDA等微型数字化产品，大到网络家电、智能家电、车载电子设备。而在工业和服务领域中，使用嵌入式技术的数字机床、智能工具、工业机器人、服务机器人也将逐渐改变传统的工业和服务方式。
　　2008年，国内仅嵌入式产品市场已经达到5707亿元的市场份额。有调查显示，消费电子、移动互联网终端、低成本PC以及嵌入式设备市场在全球将会有迅速的发展，前三项应用领域在2011年的市场都将达100亿美元，而嵌入式设备市场则将大于这一数字。在各大厂商看来，嵌入式市场没有冬天。
　　微软新军
　　嵌入式领域蕴藏的巨大市场空间将在后PC技术的时代爆发。在软件方面，Linux、Palm Os、Windows CE、风河VxWorks纷纷抢占高地，在微处理器和硬件系统的布局更是延展出长尾的影响力。掘金嵌入式市场，微软有何作为？
　　微软Windows Embedded中国区高级市场经理李涛向记者演示了这样一个场景：在利星行广场微软办公楼的一个普通会议室里，桌上放着一个像电话座机大小的一个设备，设备中间的控制面板控制着会议室里的灯光、窗帘、投影、电话、网络，里面运行的就是Windows Embedded的XP版本。
　　在微软看来，除了PC和智能手机以外，其实还有一个更广阔的领域，这个领域不像个人电脑那样，比电话和手机更加地分散，但在更广阔的专用设备的领域，用户的体验更加千差万别。
　　结合无线应用的嵌入式系统已经成为最热门的话题之一。当下的嵌入式产品正在由过去孤立的设备朝着一个连接性、服务导向的设备转变，Windows Embedded平台也从过去单纯的软件平台向“软件+服务”的平台转变。
　　2009年对于微软嵌入式业务来说是个颇为丰收的一年。2009年10月13日，微软发布新一代Windows Embedded Enterprise和Windows Embedded Server平台的RTM(release to manufacturing)版本，这是微软面向专用设备原始设备制造商(OEM)首次推出Windows7技术。手指触摸的ATM、POS机、多功能信息亭、工业电脑和医疗等嵌入式设备，未来都以Windows 7的操作系统为界面，彼此相互连接。
　　在介于智能手机与全功能的基于Windows的小型笔记本电脑间的消费类互联设备(CID)方面，微软在2009年11月6日的Tech·Ed 2009技术大会上宣布，推出Windows Embedded CE 6.0 R3的RTM(生产)版本，这个组件化的、实时的下一代Windows Embedded CE操作系统平台将帮助原始设备供应商(OEM)提供专用的个人和企业设备，为用户带来身临其境的用户体验、动态的用户界面，以及与基于Windows 7的个人电脑、服务器和在线服务间的无缝连接。
　　平台之上
　　Linux因为开放源代码而在商业机制上与一贯封闭的微软相对垒，对下游设备开发商的支持就显得尤为重要。微软的嵌入式操作系统更能够满足合作伙伴的需求：在单个工具集内实现了可靠的架构、行业标准支持、多种图形界面选项、全面的联网支持以及核心Windows技术。
　　现在全球有几百万的软件开发人员在为微软开发工具和应用程序，而且网络资源丰富。据悉，现在的Windows Embedded CE已经开放源代码，这样可以使开发商和客户有更多的控制力，以便了解他们在硬件或者软件上所出现的问题可以尽快地进行调试，其产品也就可以尽快地投入市场。而且合作伙伴和OEM在现在选择WECE软件开发时不需要先缴任何费用，只有当产品销售到消费者手中时，才需要向微软交软件授权费。
　　北京华旗资讯数码科技发展有限公司CTO范为说：“正是由于Windows Embedded CE 6.0 R3内在的特点，我们可以集中精力开发个性化的产品，例如面向Windows Embedded 的Silverlight用户界面架构和全新的嵌入式浏览器，我们相信无论是在产品中，还是在浏览器端，全新的爱国者消费类互联设备都可以让消费者享受到丰富的、引人入胜的用户体验。”
　　Windows Embedded CE 6.0 R3通过采用创新的面向Windows Embedded Silverlight 技术的用户界面框架，从而帮助设计公司及开发者间相互合作，显著改善其用户界面功能、缩短开发时间、节省开发成本以及通过基于Windows Embedded CE 6.0 R3的设备，提供独特用户体验。
　　“基于Windows 7的Windows Embedded Enterprise是用在专用设备上，可以提供OEM厂商自己的用户体验，并在虚拟化和安全性方面有很大改进，如Bitlocker硬盘锁定。而Windows Embedded Standard 2011相比于Windows Embedded Enterprise是全版本的Windows对于专用设备的授权，则是组件化的Windows，提供了多达1000多个组件。最后，Windows Embedded Server针对于专用服务器的授权，如一个大楼里对于管理摄像头服务器的应用。”李涛说。
　　掘金小屏幕
　　嵌入式操作系统市场被众多的竞争对手的产品切割得七零八碎。微软的主要竞争对手包括IBM、风河以及像Metrowerks和MontaVista Software等专门为Linux开发嵌入式版本的企业。
　　微软嵌入式系统在个人导航设备领域占有35%的市场份额，在瘦客户机、POS机市场分别占有52%和50%的市场份额，在ATM、瘦客户机方面也都占有最大的市场份额。
　　微软不断加大其在嵌入式领域投入的砝码，而中国对于微软来说不仅是潜力巨大的市场，更是重要的研发基地。这次Windows Embedded CE 6.0 R3是Windows Embedded与微软中国研发集团共同合作开发的产品，而微软嵌入式系统开发中心(MESDC)就位于微软中国研发集团内。
　　微软Windows Embedded事业部总经理Kevin Dallas表示，“微软在嵌入式领域已经有了15年的历史，从工具到操作系统到中间件到应用环境，我们在嵌入式领域已经有了非常完备的基础设施，这样的基础设施是竞争的操作系统难以企及的。对于我们来说在嵌入式领域工作的重点是，从一个单纯操作系统的平台转到综合软件+服务的平台之上，我们在这个领域做出的投资会确保这个目标的实现，包括设备的联网性。此外，我们还关注自然用户界面的基础，一旦这种自然的用户界面能够融入到设备当中就会有很多使用PC不熟悉的客户成为这种设备的客户。”
　　如果说个人电脑是微软的第一帝国的话，那么微软希望嵌入式OS是它的第二帝国。而且，在未来它将比个人电脑更为庞大。
