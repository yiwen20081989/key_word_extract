阿尔卡特朗讯收购互联网API目录服务
　　新浪科技讯 北京时间6月29日晚间消息，据国外媒体报道，阿尔卡特朗讯(以下简称“阿朗”)已经收购了开放互联网应用程序接口(API)目录服务ProgrammableWeb。
　　ProgrammableWeb收藏了超过2000个开放的互联网API，能帮助开发人员开发“混搭的互联网服务”(Mashup)。随着应用商店的流行，阿朗等公司已经意识到应用和开发者的重要性，并希望成为服务提供商和开发者之间的中介。
　　阿朗全球开发者平台副总裁劳拉·莫林(Laura Merling)表示，阿朗将进一步扩大ProgrammableWeb的目录，向其中加入Open API Service的信息。Open API Service是阿朗的一款电信网络API目录服务，目的在于整合不同运营商的服务，例如基于地理位置的服务、短信和广告，使开发者无需针对不同运营商开发不同版本的应用。
　　截至周二，ProgrammableWeb的数据库中有2042个API，其中最流行的为谷歌地图、Flickr和YouTube的接口。在该网站上，用户还可以查看4900个整合了多个不同API的网站或应用。阿朗没有披露这笔交易的条款，仅仅表示ProgrammableWeb将继续独立运营。(维金)
