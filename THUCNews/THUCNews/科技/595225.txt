苹果今日之成功源自十年前四项投资
　　导语：国外媒体周四刊文称，iTunes、Mac OS X、iPod和苹果专卖店是苹果2001年进行的四项投资，当时这似乎是一些无关紧要的产品，但10年后却成就了苹果今日之辉煌。
　　以下为文章内容摘要：
　　10年前，即2001年，苹果进行了四项投资，成就了苹果今日之成功。这四项投资在iPhone和iPad之前，是苹果在困难时期做出的。在互联网泡沫迅速席卷科技市场之际，美国经济随之衰退，苹果股价更是一泻千里。周四，苹果股价开盘于343.72美元。基于周二收盘价，苹果市值高达3172.1亿美元。而10年前，苹果股价还不足10美元。
　　这四项投资分别是：
　　1. 2001年1月，苹果推出iTunes音乐播放器。
　　2. 2001年3月，苹果发布Mac OS X 10.0(9月发布Mac OS X 10.1)。
　　3. 2001年5月，苹果开设两家专卖店。
　　4. 2001年10月，苹果发布首款iPod播放器。
　　当时， iTunes、Mac OS X、专卖店和iPod似乎是一些无关紧要的产品，但后来却成为苹果的四大支柱业务，把苹果的品牌印象从模糊变为清晰。2001年还没有像今天这样活跃的博客氛围，尽管许多科技新闻媒体报道了这些产品的发布情况，但持怀疑态度的颇多，几乎没有多少人能够意识到这四款产品的潜力。
　　2001年1月，苹果为Macintosh推出了iTunes播放器。面对Windows Media Player和RealPlayer的竞争，许多业内人士对iTunes持怀疑态度。但iTunes逐渐演变为音乐商店，后来又开始支持视频和应用。更重要的是，iTunes已经成为苹果所有内容的同步引擎。
　　2001年3月，苹果推出了Mac OS X。后来，苹果根据Mac OS X开发了iOS，如今已成为苹果最重要的操作系统。2011财年第一季度，iOS产品营收占到了苹果总营收的65%。
　　2001年5月，苹果开设了第一家专卖店。令大多数分析师不解的是，在Gateway决定退出零售市场之际，而苹果却选择进入。但事实证明，苹果专卖店是苹果展示品牌、培育Mac社区的场所。
　　2001年10月，苹果推出5GB iPod，令苹果迷大为惊喜，但并未进入主流市场。直至2004年iPod mini和2005年iPod nano上市后，iPod才正式风靡。后来，苹果开设了更多专卖店，强化了渠道，为后来的iPhone和iPad的成功奠定基础。
　　如果没有这四项投资，苹果后来的许多产品就不可能诞生：
　　没有Mac OS X，就没有iMac, MacBook/Mac Pro
　　没有iTunes和Mac OS X，就没有no iLife
　　没有iTunes和iPod，就没有iTunes音乐商店
　　没有iTunes和Mac OS X，就没有Apple TV
　　没有Mac OS X，就没有iOS
　　没有iPod、iTunes和Mac OS X，就没有iPhone
　　没有iPod、iTunes和Mac OS X，就没有苹果应用商店
　　没有iPod、iTunes和Mac OS X，就没有iPad
　　没有苹果专卖店，就没有以上任何产品
　　苹果基于这四项投资而开发出的重要产品：
　　2002年，苹果推出iLife
　　2003年，苹果推出iTunes音乐商店
　　2004年，苹果推出iLife `04
　　2005年，苹果推出iPod nano播放器
　　2006年，苹果推出首款基于英特尔处理器的Mac计算机
　　2007年，苹果推出首款iPhone手机
　　2008年，苹果推出应用商店(App Stor)
　　2010年，苹果推出iPad平板电脑
　　2011年，应用程序下载量突破100亿次，又推出了Mac应用商店。
　　除了这四款产品，还有另外一款产品对苹果的成功起到了重要作用，即2003年推出的Safari浏览器，如今已成为苹果多款产品的必备应用，如iTunes、iPad、iPhone和iPod touch等。(李明)
