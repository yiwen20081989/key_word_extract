仅重0.725Kg 全球最轻上网本超高价发布
    NEC于7月下旬即将推出最新型号的上网本机型“Versa Pro J UltraLite Type VS”，由于其重量只有752g，厚度只有15.8mm，故此被喻为全球现时最轻薄的上网本，售价为99000日元，折合人民币约为7000人民币。
    据NEC官方介绍，为令Versa Pro J UltraLite Type VS造得如此轻薄，NEC为其配备了低功耗的Intel Atom Z540处理器（核心频率为1.8GHz），内建Intel GMA500绘图芯片，更因减薄机身而用上10.1?1280x720像素解像度的LED背光屏幕。
    虽然产品轻薄，但仍加入NEC常用于常规笔记本的保护功能，跟很多旗下产品一样可以承受150kg的压力。此外，产品配备1GB DDR2内存，以及令系统运行更快且更安全的64GB SSD固态硬盘。扩充方面，机身共设置有3 组USB接口。
    操作系统方面，NEC Versa Pro J UltraLite Type VS可选择配备Windows Vista Business或 Windows XP Home SP3操作系统。由于Versa Pro J UltraLite Type VS主要针对高流动性用家市场，故此相信应支持无线网络，以及有不俗的电池续航能力，其它详情仍有待公布。
