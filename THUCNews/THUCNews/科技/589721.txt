微软一季营收129.2亿美元 净利35.7亿
　　新浪科技讯 北京时间10月23日晚间消息，据国外媒体报道，微软今日宣布，截至2009年9月30日的第一财季营收为129.2亿美元，同比下滑14%。当季运营利润为44.8亿美元，净利润为35.7亿美元，稀释后每股收益40美分，同比分别下滑25%，18%和17%。
　　微软当季业绩还包括了一笔14.7亿美元的递延营收，对稀释后每股收益的影响为12美分，这笔营收与 Windows 7 Upgrade Option项目以及微软向OEM(原始设备制造商)和零售商销售Windows 7的营收有关。若包括这笔递延营收，微软当季营收为143.9亿美元，同比下滑4%，当季每股收益52美分，同比增长8%。
　　微软CFO克里斯·里德尔(Chris Liddell)说：“我们对公司第一财季的业绩感到非常满意，尤其是对Windows获得的强劲消费者需求感到高兴。我们还维持了严格的成本纪律，这使得微软在持续困难的整体经济状况下依然能够实现强劲的收益表现。”
　　微软在10月22日在全球推出了Windows 7以及Windows Server 2008 R2。微软还在10月推出了Microsoft Exchange Server 2010，在7月宣布与雅虎就网络搜索业务达成战略合作关系。
　　微软COO凯文·特纳(Kevin Turner)说：“在全球发布Windows 7、Exchange Server 2010以及Windows Server 2008 R2是微软、我们合作伙伴和用户的一个重要里程碑。我们很高兴这些产品获得了积极的初步反应。”
　　微软宣布，截至2010年6月30日的当前财年运营支出预期从265亿美元下调至262亿美元。
　　微软管理层将在美国太平洋标准时间周五上午7:30(北京时间周五晚上10:30)召开分析师会议，讨论第一财季业绩以及公司的业绩前景。主持会议的包括微软高级副总裁兼CFO里德尔、企业副总裁兼首席会计官弗兰克·布罗德(Frank Brod)、负责投资者关系的总经理比尔·科夫德(Bill Koefoed)。投资者可以访问微软网站http://www.microsoft.com/msft收听此次会议的网络直播，网络录音将保留至2010年10月23日。(逸飞)
