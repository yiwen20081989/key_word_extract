年底前有15厂商采用Windows Mobile 6.5
　　【赛迪网讯】9月17日消息，昨天，2009年中国国际信息通信展览会在中国国际展览中心新馆正式开幕。微软大中华区电信及媒体事业部移动产品组高级总监陈敬新先生接受了赛迪网的现场高端专访。
　　陈敬新现任微软公司大中华区电信及媒体事业部移动产品组高级总监，全面领导Windows Mobile的销售和业务拓展工作。陈敬新负责推进与价值链中重要合作伙伴之间的商业关系和战略合作，其中包括手机制造商、分销商、零售商、以及主要的移动运营商，从而推动Windows Mobile和微软移动通信服务在中国大陆、香港和台湾的业务收入。
　　在此之前，陈敬新曾担任微软公司移动通信事业部大中华区总监，负责Windows Mobile业务在大中华地区的全面策略制定、规划和市场营销工作，并且在将微软中国移动通信事业部发展成为微软全球增长最快并且最成功的业务部门方面扮演了至关重要的角色。
　　陈敬新在亚洲地区的计算机软件和通信行业拥有超过十年的工作经验，在销售、业务拓展与市场营销方面积累了丰富的经验。
　　在来到中国之前，陈敬新曾于2005年至2007年初在日本东京工作，担任诺基亚移动软件部门亚太及大中华区高级经理，负责诺基亚基于Symbian的S60平台在整个亚洲地区的市场营销工作。
　　在此之前，2000年至2005年，陈敬新在微软新加坡工作，最初担任技术专员，负责推动微软客户端技术的采用和销售工作，而后被晋升为解决方案市场经理，负责推动微软公司两个最大的并且利润率最高的部门，即Windows和 Information Worker部门的业务收入和市场份额增长。在加入微软公司之前，陈敬新曾成功创建经营过一家IT服务公司，直到今天，他还保持着企业家的素质。
　　敬新拥有美国芝加哥大学颁发的工商管理硕士学位，以及新加坡国立大学颁发的市场营销专业工商管理学士学位。
　　观点摘要：
　　年底前共有15个厂商的30多款手机将采用Windows Mobile 6.5
　　Windows Mobile开放并不开源，确保用户体验和平台的一致性
　　Windows Mobile和OPhone拥有共同的竞争对手，那就是非智能手机
　　希望5年内非智能和智能手机之间的界限模糊
　　合作伙伴和竞争对手并没有绝对的定义，微软和诺基亚有激烈的竞争也有紧密的合作
　　如果PC厂商推出其他智能系统的手机，我们希望用户会问：这是什么？为什么不是Windows Phone？
　　之前太过于专注对PC非常熟悉的客户，但Windows Mobile 6.5迈了一大步
　　赛迪网：这次见到您职位有一些变化，请问您现在的工作职责是什么？和之前有什么不一样？
　　陈敬新：从微软中国移动通信事业部大中华区总监到大中华区电信及媒体事业部移动产品组高级总监，其实并没有太大的分别，我们还是把移动通讯领域作为初衷。
　　赛迪网：今年包括天语、中兴在内的许多国内手机厂商都发布了基于Windows Mobile的智能手机，请问现在微软和多少个国内厂商合作推出过智能手机？大概有多少款？
　　陈敬新：简单统计一下，截止到今年12月底，即将采用我们新手机操作系统Windows Mobile 6.5的会有超过10个国内厂商推出20多个型号的手机。如果把国外厂商加进来的话，总共会有超过15个厂商的30多个型号。
　　赛迪网：现在智能手机系统也比较多样化，您认为这些国内手机厂商都一致选择了微软Windows Mobile是什么原因？
　　陈敬新：我希望分享的是大家前一段时间比较留意一种开放的合作形式，其实微软一直以来也非常开放，这一点我们和一些新的竞争对手是没有区别的。我们在过去6年有超过125个开放的API，开发商可以发挥自己的创意，在Windows Mobile平台上进行开发是完全免费的，开发商可以创造自己的市场。
　　区别在于我们不开源，这样就可以确保平台的一致性，包括用户体验、平台的兼容性都不会改变，这比起开源系统来说是更有好处的。举例来说，虽然有同样的“地基”，但中国联通推出的Android和移动推出OMS将会不同，二者也并不能兼容。基于我们的理解，如果系统之间无法兼容就不能称之为是同一种系统。
　　我们认为Windows Mobile非常广阔，同时也很丰富。从广阔来说，我们拥有很大的开发群体，全球基本上有70%的人都已经可以在Windows Mobile上进行开发。使用Windows Mobile的终端有不同的价位、品牌和设计，用户可以根据自己的喜好选择品牌，以此享受Windows Mobile给他带来的便利和好处，我们的合作伙伴也非常愿意在Windows Mobile上进行投入。同时，我们也很丰富，因为我们有多年运行累积起来的各种优势，包括移动办公软件、非常完善的浏览器以及丰富的第三方软件程序。
　　赛迪网：随着3G和智能手机越来越被大家熟知，许多国产手机厂商表现也非常抢眼。您认为国产手机厂商是否能借此机会有一个升华？
　　陈敬新：我非常希望国内厂商能有一个好的前途。我从3年前搬到北京来之前就非常留意中国手机行业以及TD的发展，特别是今年恰逢60大庆，我也十分荣幸可以参与到这个行业的发展。
　　赛迪网：那么您如何看待微软和国产手机厂商这样一种合作的关系？
　　苹果的优势是硬件和软件同时都由自己来做，以此控制了生态链中的种种环节。我们也长时间坚持了自己的基本原则，那就是微软的优势在于提供软件，让硬件厂商发挥自己的优势来提供硬件，这样整个产业链得以互相竞争。经过长时间竞争之后我们已经看到结果，那就是Windows占据了绝对的优势。
　　我们认为手机终端的环节与PC其实没有太大的不同，也是产业链之间的竞争。从微软的角度来说，我们还是非常坚持我们长时间累积起来的竞争优势，同时我们也愿意让生态链的各环节都发挥他们自己的优势。因为我们认为最好的前景就是这样来创造的，我们相信共赢。
　　赛迪网：我们知道微软和中移动一直都有非常好的合作关系，但近日中移动刚刚推出自己的应用商城和OPhone系统，请问这是否会影响微软和中移动的关系？
　　陈敬新：微软之前和移动有非常好的合作关系：移动飞信平台的后台是基于微软的技术；在TD发布前，我们已经有了三星I688和多普达S700；TD正式发布后，我们还和LG推出了首款深度定制的Windows Mobile手机KT878，这些都是既定现实，不会改变。近日宏达电和移动之间有协议提供一系列产品，其中即将发布的第一款也是基于Windows Mobile的智能手机。
　　中国移动对自己的定义是一个运营商，而不是软件供应商，它发布OPhone的初衷也是为了促进整个行业的发展，而且也公开表示推OPhone并不代表排斥其他平台。我们认为，如果用户能够有更多的选择是一件好事。中国移动拥有足够庞大的客户群体，我们和OPhone拥有共同的竞争对手，那就是非智能手机。
　　赛迪网：有数据称现在传统手机销量仍占手机总销量的72%，您认为智能手机多久能够成为主导？
　　陈敬新：我们希望在5年内非智能手机这个词汇完全消失，这里并不是说非智能手机的市场消失，而是我希望在5年以内非智能手机和智能手机之间的界限已经模糊。
　　我们认为智能手机与非智能手机相比起来具有绝对优势，因为随着行业的演变和发展，计算平台所要求的肯定不仅仅是厂商的需求，而是三方整体的生态系统能够提供什么，而操作平台之所以存在就是要支持三方的软件和服务。
　　在Android、Symbian、Mac OS等众多竞争对手中，微软希望在智能市场上占据多大的份额？
　　我们重视出货量的增长率和合作伙伴、合作范围的增长。刚刚提到，在年底之前我们仅仅只在Windows Mobile 6.5上就有15个厂商的30款手机，对于这个数量我们还是非常满意的。
　　赛迪网：8月12日，微软与诺基亚结成联盟引起了很大的关注。因为微软自己有智能手机系统Windows Mobile，而诺基亚作为全球最大的手机制造商主要是采用Symbian智能系统。这是否说明微软和诺基亚在智能手机系统上将由竞争转为合作？一致对抗其他竞争者？
　　陈敬新：可以这样来看，微软和诺基亚也有非常紧密的合作，但同时我们Windows Mobile和诺基亚S60也有非常激烈的竞争，我们这个行业在发展的过程中合作伙伴和竞争对手并没有绝对的定义。
　　诺基亚和微软的合作不仅仅限于移动终端，因为诺基亚也在推上网本，就是采用 Windows 7系统，它涉及的这个领域我们可以给它提供支持。在智能手机产品，如果诺基亚在短期内不直接授权给Windows Mobile，我们在短期内可以给它的就是OfficeMobile。
　　赛迪网：诺基亚涉足上网本，那您如何看待像华硕、宏?、戴尔这样的PC厂商涉足通信行业推出智能手机？是否会担心这些厂商选择其他的智能系统？
　　陈敬新：Windows Mobile一向的优势就是和Windows品牌很接近，我们接下来定义的Windows Phone就跟Windows PC很接近，在数据同步等体验上都是非常一致。如果PC厂商推出其他智能系统的手机，我们希望用户会问：这是什么？为什么不是Windows Phone？
　　赛迪网：首款使用Windows Mobile 6.5的手机将在10月6号推出，大家都非常期待，请问这会给手机用户带来怎样的体验呢？是否今后采用Windows Mobile 6.5的手机都会被称作是“Windows Phone”？
　　陈敬新：简单来讲，操作系统的名字还是Windows Mobile，但是整个产品类别我们定义为Windows Phone。这其实和Windows PC定义的名称差不多，因为我们都会说Windows PC而不会说是Windows Professional PC。
　　Windows Mobile 6.5在用户体验方面有非常大的改变和进步。我们之前收到过关于Windows Mobile 6.1和之前版本的反馈，这些反馈表明我们之前太过于专注对PC非常熟悉的客户，。这样就会使得很多手机用户使用Windows Mobile之后，会要求它更像是手机而不是PC。Windows Mobile 6.5向这方面迈了一大步，操作界面很熟悉，也更加友善，既保有功能，又有更漂亮的外表，使用上也更加简单。
　　赛迪网：通信展有何亮点？初步印象？
　　陈敬新：就我在短期内看到的印象就是中国移动通讯这个行业在持续不断的发展。我看到今年酷派有很大的展台，虽然展示的而是基于Windows CE自己开发的软件平台而非Windows Mobile，但我们在短期内也会宣布和酷派有一些合作。我也看到三星展台上既有智能手机，也有非智能手机，并且它在短时间内也会发布旗舰级别的 Windows Mobile产品。
