第一财经周刊：HTC的直觉
　　通过用户自定义交互系统，HTC为Andriod高端手机市场增添了更多价值。
　　文｜CBN记者 邹?
　　苹果粉丝伤不起。白色iPhone4雷声大雨点小，而Andriod手机一部接着一部地出。4月13日，HTC发布了一款被称为Sensation的“超级多媒体手机”，其配置据说是今年 HTC 高端机的标杆。
　　最直观的数据是Sensation搭载了高通(Qualcomm)Snapdragon 1.2 GHz 双核处理器，反应速度更快；4.3 英寸大屏幕，分辨率达到960 x 540；虽然摄像头只有 800 万像素，但具备了1080P的高清录像能力，值得期待。
　　在外观上，Sensation后壳设计元素与去年HTC的 Windows Phone 系列一脉相承，都采用铝合金材质。有趣的点在于后壳的不对称设计，隔很远也能一眼认出这是Sensation。
　　HTC为Sensation配置了新版本的 HTC Sense UI，这个新交互系统可以让手机在锁机状态下展示更多信息。不用滑开那个解锁键，用户就可以看到短信、邮件、SNS 更新、天气，股市等信息。解锁后，用户可以直接进入短信、邮件或者拍照等应用，点击图标的步骤被大大减少。
　　在分化严重的Andriod手机市场，HTC的实力不可小觑。美国市场调查机构Social Nuggests指出，经过分析一百万个社区媒体上的对话，HTC 旗下的Thunderbolt不但超越iPhone成为当下最被热烈讨论的科技产品，HTC一系列智能手机更一举囊括这项排行的前三名，远远超过位于第十名的iPhone4。
　　这于Android 手机市场而言是个突破性的好消息。尽管它们整体的销量已经超过 iPhone，但从来没有单独哪款产品表现超过 iPhone。
　　苹果的魔力在于它总能留住用户，HTC在这一点上比其他Android 平台手机品牌找到了更具新意的策略。HTC没有把力气花在开发自己的操作系统上，而是开发了被称为“用户体验层”的Sense UI。技术团队认为，不执着于解决技术问题，只搭一个框架，让用户自定义才是最好的解决方案。
　　这个名为”直觉“的交互系统还兼容 Windows Phone 7、Windows Mobile 和 Brew不同平台的产品─HTC抓住了Android开放的精髓。
　　Sense UI开发基于细致的用户观察，比如用户从包里拿出手机的时候铃声会变轻。而未来用户体验不仅在手机上，手机与其他产品的交互，例如电视、机顶盒和PC之间的兼容也会成为HTC的关注点。
　　“我们享有自由，而不是某种特有的操作系统。我们做了一个‘用户体验层’，专注于此可以带来最大的价值。”HTC 用户体验负责人 Drew Bamford说。
　　Sense UI设计的背后
　　优先性。Sense UI设计团队根据消费者调研发现用户第一重视的元素是电子邮件，第二位则是天气。
　　跨平台。每次他们做设计，团队都需要考虑到这一点：Sense UI 要覆盖多种操作系统—目前是 Android ，WIndows Phone 7、Windows Mobile 和 Brew 。规格和价格也有高中低之分，从最顶级的 Android 旗舰机 HTC Desire HD ，到最普通的 Windows Mobile 低端机 HTC HD mini。
　　自定义。在许多方面，HTC Sense 像是一个空的容器，用户能够按照不同的方式来使用它。比如，Sense 能保存不同的场景，用户可以自定义，也可以通过网站下载喜欢的场景。HTC 用户体验负责人 Drew Bamford说：“我们定制的框架很好，能容纳各种东西，适应各种用户。”
　　抹除差异。Sense UI的设计哲学是“这是一个任由使用的工具，来创造你自己的体验。”因此消除操作系统间的差异变得很重要，让不同平台用户之间流畅使用交互界面从来不是简单的事情。
　　用户洞察。画画图，或上上色，其实只是HTC复杂设计过程的最后一步。Drew Bamford表示HTC Sense UI 第一步也是最关键的一步，是深刻理解客户。“客户需要一部怎样的手机？这种研究占用了我们大部分的时间，却常常被外人所忽略。”Sense UI的定位是“即便你从来没用过手机也可以马上上手”。
