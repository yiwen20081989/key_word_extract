黑莓上数据库文件的说明
　　对你最重要的永远是数据，好比我们常说的，手机丢了不可怕，里面的联系人丢了才是大麻烦。一起来学习一些基本只吧。说明中文翻译by berrytimes。英文原文说明看源地址在这里。这些信息尤其在你进行备份和恢复的时候会经常看到，了解以后你可以大致清楚备份恢复里面每一条对应的都是你的手机里面的什么数据，比如对于绝大多数人来说，备份你的黑莓，Address Book、SMS message可能是最最重要的两条数据了。Database Functionality Address Book 保存联系人数据 Address Book Options 保存联系人设置数据 Alarm Options 保存闹铃数据 Attachment Data 保存手机上查看过的附件的数据 Attachment Options 保存附件浏览器用户自定义的数据 AutoText 包括了自动文本替换的定义数据，比如：adn > and. Browser Bookmarks 包括了官方浏览器中书签的数据 Browser Data Cache 包括浏览器的缓存数据 Browser Folders 允许书签保存到不同文件夹分类 Browser Options 包括浏览器设置信息 Browser Push Options 包括了BlackBerry Browser的push内容设置 Browser URLs 包含了以前你输入的URL的列表信息 Calendar 保存了日历的内容 Calendar Options 保存了日历的设置 Categories 允许给联系人、约会、人物和记事本进行分类 Content Store 包括了手机中图片和其他信息 CustomWordsCollection Default Service Selector保存了7100系列手机的自定义列表 定义默认的消息服务Email Filters 对收到信息过滤的设定 Email Settings 定义手机首发信息的方式 Firewall Options 开启或者关闭防火墙文件夹id定义哪个文件夹用于存放信息 Folders 讲用户自定义的inbox文件夹进行列表 Handheld Agent 包括了关于手机的诊断信息 Handheld Key Store 安全储存了数据联络和签名的编译密码键值 Key Store Options 用于设置手持键保存的设置 MemoPad Options 保存记事本的设置 Memos 保存记事本内容 Message List Options 定义Message界面中信息的显示列表方式 Messages 包括所有的信息内容，短信，彩信，邮件等。 Options 保存手机的全部设置信息数据 Phone Call Logs 呼入打出电话的全部信息数据 Phone Hotlist 保存手机中电话号码的列表 Phone Options 保存手机设置 Policy 保存手机的IT Policy(IT策略) Profiles 包括所有情景模式信息，比如新的日历约会的震动，或者是新消息到达的铃声等。 Profiles Options 选择手机提醒的当前情景模式 Purged Messages 包括了手机上删除了的信息的查询 Quick Contacts 保存了快速拨号列表里的电话号码 Random Pool 保存安全为编译码生成随机号码的号码。 Recipient Cache 包含为联系你的联系人的编译情景模式 Ribbon Bar Positions 保存你桌面图标的排列方式信息 RMS Databases 保存你注册程序的信息 Service Book 保存为了让手机链接不同服务而提供的SeviceBook信息 SMS Messages 保存发出和收到的短信内容 Tasks 手机中列表和任务内容 TLS Options 设置传送层的安全设置 Trusted Key Store 包含手机的信任键值 WAP Push Messages 保存WAP push内容 WTLS Options 保存无线传输层安全的设置
