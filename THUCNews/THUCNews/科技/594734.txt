苹果将把Mac应用下载由网站转移至应用商店
　　新浪科技讯 北京时间12月21日晚间消息，苹果高管周一表示，在Mac App Store应用商店推出之后，将于当天把Mac OS X系统的应用下载从网站转移至该应用商店中。
　　苹果开发者关系主管罗恩·冈本(Ron Okamoto)周一致信苹果的注册开发者。他表示：“我们认为，Mac App Store将是用户发现、购买和下载应用的最好场所，因此我们将不再通过Mac OS X Downloads网站提供应用。从1月6日开始，我们将引导用户通过Mac App Store获取多种应用。”
　　Mac OS X Downloads目前是苹果网站重要的一个栏目。此前，用户可以通过操作系统菜单的“Mac OS X Software”选项登录该网站。苹果于上周宣布，将于1月6日推出Mac App Store服务。
　　周一，罗恩·冈本呼吁开发者将关注重点转向苹果新的应用商店。他表示：“我们感谢你们对Mac平台的支持，希望你们利用这一新的机会向更多用户展示你们的应用。”
　　由于苹果针对Mac App Store设定了一些限制，因此当前Mac OS X Downloads网站上的一些应用将无法进入该应用商店。例如，目前在网站上排名第4的系统工具ClearMyMac将受到限制。ClearMyMac是由乌克兰开发商MacPaw开发的一款系统管理工具。该工具能够禁用一些系统应用，因此与苹果的规定相冲突。
　　另一款当前排名第9的应用Snow Leopard Cache Cleaner也可能受到限制。该工具帮助用户定制Mac OS X用户界面的一些元素，包括禁用控制面板及内建的Spotlight搜索等。苹果对Mac App Store的规定显示，任何应用不得改变系统的原生用户界面，以及系统的行为。
　　而另一款苹果目前主推的应用Autodesk Sketchbook Pro 2011也将遭禁。这款软件目前在网站上提供测试版下载，完整版的售价为80美元。而根据苹果的规定，开发者不得在Mac App Store中提供测试版或演示版应用，该应用商店只接受功能完整的零售版应用。
　　苹果取消Max OS X Downloads的另一个副作用在于，使用较老版本Mac OS X系统的用户将较难找到应用。苹果的规定显示：“不运行在当前版本系统上的应用将被拒绝。”Mac App Store将支持Mac OS X 10.6“雪豹”系统，而不支持10.5和10.4。而目前苹果网站上提供的一些应用能够运行在多个版本的系统上。
　　如果希望在Mac App Store推出之初就提供应用下载，那么开发者应当在美国太平洋时间12月31日17:00之前向苹果提交应用。(维金)
