微软承认Windows 7补丁存在故障
　　【赛迪网讯】11月11日消息，据国外媒体报道，Windows 7用户反映，当跳转到操作系统的最新补丁包后，会遇到有关TIFF文件的故障，微软(微博)公司已经承认了这一问题，并为受影响的客户提供了可下载的补丁。
　　据报道，在具体的故障案例中，在Windows 浏览器中显示TIFF图像文件的缩略图需要很长时间。微软解释：Windows 7 补丁包(SP1) 用户和Windows Server 2008 R2 补丁包(SP1)用户会受到影响。照此情况看来，Windows 7 正式版和Windows Server 2008 R2 正式版都将受到影响。只有当用户激活Windows浏览器打开包含有TIFF的文件夹时，才会出现这个问题。TIFF代表标签图像文件格式，专门用于数字图像，使用频率远远低于JPEG，只有当需要质量比联合图像专家组版式更好的图片时，用户才会使用TIFF。微软说，如果文件很大或者存储在网络共享中需从Windows 7 SP1机器中读取，Windows 浏览器显示TIFF图像的缩略图的速度就很慢。微软公司为解决此问题已经向用户提供了KB2619291热修复补丁。为了配置这款补丁，用户需要运行Windows 7 SP1或者Windows Server 2008 R2 SP1.KB2619291将会进入Windows 7 SP1和Windows Server 2008 R2 SP1，然后释放给所有用户，但目前用户需要自行下载和安装。
