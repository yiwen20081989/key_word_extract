苹果招聘工程师为Mac OS X开发革命性功能
　　新浪科技讯 北京时间7月30日午间消息，据国外媒体今日报道，苹果的最新招聘信息显示，该公司正在招聘工程师，为Mac OS X开发“革命性”的功能。
　　苹果在本周发布的招聘信息中写道：“你想参与创建一些新东西吗？一些从未有人做过而且肯定能够令所有人吃惊的东西吗？如果你参与创建的东西每天都能被数百万苹果用户使用，你会感到兴奋吗？那么就请加入我们的Mac OS X软件工程师团队，参与创建一款全新的革命性功能。”但苹果并未透露具体功能信息。
　　苹果希望招聘一名拥有互联网技术和服务开发经验的软件工程师，“优秀的候选者”需要拥有HTTP协议以及大型网络系统架构的开发经验。
　　业内人士此前预计，苹果将在今年夏天的全球开发者大会(以下简称“WWDC”)上发布最新的Mac OS X 10.7。至少也会进行一些演示，但本届WWDC却完全被iPhone 4占据。
　　为了开发这些神秘的新功能，苹果希望希望聘请一名高级软件工程师帮助创建一款软件，并将其作为Mac OS X“根基”的一部分。
　　苹果在招聘启事中说：“我们正在开发一些真正革命性且绝对令人振奋的东西，而且需要你最具创新力和专注性的努力。”该公司希望招聘一名有动力克服“非常困难”且“从未有人做过”的挑战。
　　该职位的理想候选人需要拥有计算机科学学位，以及5年的C/C++/ Objective-C或用于用户系统的框架的专业开发经验。申请者最好能够具备Mac OS X和Unix开发经验。
　　就在Mac OS X 10.6雪豹去年发布后，便有证据显示苹果开始开发Mac OS X 10.7系统。当时，负责Mac OS X的开源“launchd”框架的一个新数据库入口提到了“11A47”。这个字符串代表了苹果Mac OS X的版本号，表明这是Mac OS X 10.7的早期版本。(鼎宏)
