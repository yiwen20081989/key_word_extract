Chrome Web Store全球发布 谷歌仅提成5%
　　新浪科技讯 北京时间5月12日上午消息，谷歌今天在2011年I/O开发者大会上宣布，支持41种语言的Chrome Web Store面向全球发布，让应用开发商能接触更多用户并从中谋利。
　　谷歌Chrome业务高级副总裁Sundar Pichai称，Chrome Web Store目前已经向全球用户开放，可支持41种语言。他还指出，自2010年12月份在美国推出后，Chrome Web Store的应用安装已达1700万。
　　谷歌工程师Vikas Gupta还称，由于太多的应用选项导致收费结构过于复杂，故他们一直都在对Chrome应用程序接口(API)进行升级。目前开发商可将点击支付和单行代码合并到应用中，且谷歌应用商店分成仅为5%。这一新的定价模式意在让开发商在Chrome平台建立“令人惊讶的业务”。
　　另外，谷歌工程副总裁Linus Upson称，公司计划让Chrome Web Store支持火狐等更多浏览器，但未透露具体详情。(姜雪)
