微软警报称Windows新漏洞可能被黑客利用
　　新浪科技讯 北京时间1月5日早间消息，微软表示，部分版本Windows操作系统的一个漏洞可能被黑客利用，从而使他们可以远程控制用户的个人电脑。
　　微软是在周二的一次特别警报中发布这一信息的，并表示，尚未开发出修补这一漏洞的补丁，并且尚未找到绕过这一问题的方法。
　　微软表示，目前还没有发现任何试图利用这一漏洞进行的攻击，该漏洞涉及Windows XP、Windows Vista、Windows Server 2003和Windows Server 2008的图形处理机制。
　　据悉，这一问题不会影响到微软的最新PC操作系统Windows 7和最新服务器操作系统Windows Server 2008 R2。(钦亮)
