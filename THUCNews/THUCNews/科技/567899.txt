谷歌发布Chrome OS操作系统 明年全球逐步推出
　　新浪科技讯 北京时间12月8日凌晨消息，谷歌今天在美国旧金山举行媒体发布会，发布有关Chrome OS操作系统的最新情况。
　　以下为发布会直播实录：
　　2:31：谷歌产品管理副总裁桑达·皮采(Sundar Pichai)上场。
　　2:33 皮采称，ChromeOS最早发布于2008年，当时正是AJAX和Web 2.0的革命时代。
　　2:34 皮采说，“Chrome浏览器在我们举办Google I/O大会的时候拥有7000万活跃用户，但今天，其活跃用户已达1.2亿人。”
　　2:36 他首先展示了谷歌的即时搜索功能。他表示，谷歌即时搜索已经被整合到Chrome的地址栏(omnibox)中，用户不需要进入Google.com，就可以进行即时搜索。
　　2:39 他展示了Chrome浏览器的内置PDF阅读插件。
　　2:40 他正在展示Chrome的硬件加速功能。画面中是一个使用WebGL技术显示的水族馆。他说，“所有这些都由GPU进行渲染。”
　　2:44 桑达·皮采重返舞台，谈论V8 Javascript引擎。
　　2:45 皮采说，“Chrome是最快的，比IE快16倍。我们一直在努力提升Javascript性能。”
　　2:46 皮采说，“今天我们为V8引擎作出了一项称为Crankshaft的改进，使得速度比此前增加了两倍。如果与两年前的IE相比，我们的速度要快上一百多倍。”
　　2:47 皮采在谈论浏览器的简洁、Chrome的自动更新功能，以及Chrome Sync同步功能。Chrome Sync现在已经能够同步所有的数据，包括书签、扩展等等。
　　2:52 皮采在谈论Chrome的沙盒功能。沙盒功能将标签保护在沙盒之中，现在还可以将第三方插件运行于沙盒之中。
　　2:52 现在皮采开始谈论Chrome Web Store。他说，“我们的目的是帮助用户发现这些优秀的应用。”
　　2:54 皮采说，“用户想要付费使用应用，但他们不愿相信小的、独立开发者。这正是Chrome Web Store试图解决的问题之一。”
　　2:54 Chrome Web Store页面的左侧是应用分类，右侧是推荐应用。
　　2:56 Chrome Web Store目前支持Chrome 8浏览器。该商店与用户谷歌账户绑定，每次购买都将记录在账户之中。
　　2:58 皮采展示了一款简单的2D“图片找茬”游戏，展示Chrome运行游戏的能力。
　　3:00 纽约时报数字运营首席技术官马克·弗朗斯(Marc Frons)上场，介绍纽约时报的Chrome应用。
　　3:03 马克·弗朗斯展示纽约时报的Chrome应用，但现场的Wi-Fi出了一点问题。
　　3:03 EA游戏高管登台展示一款名为Poppit的游戏。
　　3:04 EA高管说，“我们在不到48小时的时间里就将游戏转换成为一款动态HTML 5 web应用。这是我们制作得最快的Poppit。”
　　3:05 Poppit游戏将绑定在Chrome 9中，用户甚至不用动手下载游戏。
　　3:08 两名亚马逊副总裁Eva Manolis和David Limp登场介绍“亚马逊橱窗”(Amazon Windowshop)应用。“该应用已经在Chrome Web Store中上架。”该应用支持触摸屏界面，用户可以在应用中浏览产品图片。
　　3:11 亚马逊发布Kindle for the Web应用，进一步实践“一次购买，随处都可阅读”的Kindle理念。图书浏览界面拥有类似于Cover Flow的风格。
　　3:14 皮采重新登场。他表示，Chrome Web Store今天已经在美国上线，地址为chrome.google.com/webstore。“Chrome Web Store首先在美国上线，并将于明年一季度逐步在全球推出。”
　　3:15 皮采开始谈论Chrome OS操作系统。新系统的口号是，“一切皆在网络”(Nothing but the web)。”
　　3:17 皮采表示，演示所用硬件仅为“参考硬件”。现在，他将为大家演示Chrome OS操作系统的整个设置过程。
　　3:19 皮采正在展示运行Chrome OS系统的笔记本。
　　3:20 现在是运行Chrome浏览器的PC和运行Chrome OS操作系统的笔记本的对照演示。
　　3:20 首先演示的是两台机器间的实时书签同步功能。
　　3:23 Chrome OS还拥有匿名模式，在用户与他人共同使用设备时保护其所有个人行为。
　　3:25 皮采将笔记本的网络切断。此时，正在运行的谷歌文档就进入了离线模式，用户可以继续在断网的情况下进行工作。皮采表示，这一功能也将出现在普通的谷歌文档服务中。
　　3:26 皮采说，“我们做了大量的工作，确保用户能够选择以何种方式使Chrome上网本连接网络，”他点击了一个按钮，将上网本切换到蜂窝网络模式。该台上网本此时显示拥有4格3G信号。
　　3:27 皮采在演示使用谷歌云打印(Google Cloud Print)打印一份食谱。
　　3:27 皮采说，“我们与Verizon合作，为每一部Chrome OS笔记本提供无缝地网络连接。”
　　3:28 用户可以购买Chrome OS笔记本并签约Verizon，每月获得免费的100M数据流量。
　　3:29 用户还可以选择9.99美元起步的预付费计划。皮采表示，“用户当然也可以离线使用Chrome OS笔记本，但我们在每一部Chrome OS笔记本中都采用了高通Gobi全球通用调制解调器，”从而支持全球的网络，使得用户可以随意移动，随时保持网络连接。
　　3:31 既然谈到了无线网络，我们回过头来谈谈安全问题。与Chrome浏览器一样的是，Chrome OS操作系统也拥有自动更新和沙盒功能。同时，Chrome OS还支持数据加密。
　　3:33 Chrome OS还拥有验证启动功能(Verified Boot)，即以加密的方式检查修改后的系统，如果发现任何错误，用户可以恢复到先前版本的Chrome OS。
　　3:33 用户没有root权限。
　　3:35 皮采正在谈论Chrome OS笔记本多么地适合企业用户。虚拟化厂商Citrix Systems高级副总裁戈登·佩恩(Gordon Payne)登场，谈论其与谷歌在Chrome OS项目上的合作。
　　3:43 皮采重新登场。他表示，他在过去的6个月里一直在使用一台Chrome笔记本。“Chrome OS一直在不断优化。”
　　3:45 皮采表示，目前Chrome OS相关项目仍然处于测试阶段，但Chrome OS会变得越来越好，更多的功能都在开发之中。“我们希望用户能够插入相机。我们还没有做这方面的工作。”
　　3:46 皮采正在向大家展示Chrome OS早期项目的原型机。
　　宏?、三星和英特尔是谷歌Chrome OS的硬件合作伙伴。预计搭载Chrome OS操作系统的硬件将于2011年年中发布。
　　3:47 皮采说，“Chrome OS笔记本将拥有多种颜色。我手中的这台黑色笔记本唯一的作用只是测试软件。”
　　3:48 这台笔记本拥有12.1英寸显示屏、全尺寸键盘、全球漫游3G、大号触摸板，以及网络摄像头和802.11n双频段Wi-Fi。它拥有8小时以上电池寿命，待机时间超过8天。
　　3:50 CR-48先锋项目将在部分企业启动运行，如美国航空公司将使用Chrome OS设备处理机票预订工作，另外还有部分酒店呼叫中心也将使用这些设备。
　　3:52 皮采说，“一部分用户也将有机会参与这个项目”。但他没有说明参加项目的方式。
　　3:53 谷歌CEO埃里克·施密特(Eric Schmidt)出场，对Chrome OS团队表示祝贺，发表演讲。
　　3:53 埃里克·施密特说，“谷歌是全球为数不多的能够大规模进行真正的计算机科学研究的公司之一。”
　　3:56 施密特谈到了他早期在Sun的情况。在谈到瘦客户端的时候，他说，“为什么瘦客户端如此难以实现？”
　　3:56 施密特说，“为什么Sun会失败？为什么你们现在应该相信谷歌？在Sun时期，我们对这一问题的理解对错各占一半。我们当时没能真正认识到问题的复杂程度和微妙性。”
　　3:56 施密特说，“根据摩尔定律……磁盘运行速度如今已经大大加快，网络也更加可靠……”换句话说，科技进步已经追赶上了理念的进步。”
　　3:56 “因此，软件技术也需要跟上。“所有这些工作都得以完成，Chrome这个现代化的浏览器才能出现。”
　　3:59 施密特表示，当初谷歌创始人拉里和佩吉提出开发一款浏览器的想法时，他坚决反对。
　　4:00 施密特说，“我们终于有了一款规模足够大、技术含量足够高、速度足够快的产品，我们可以在这个产品上建立新的基础架构。”
　　4:01 施密特说，“我们十年前或二十年前的直觉是对的，但当时我们没有必要的技术。”
　　4:02 皮采重新登场。以下为问答环节。
　　4:04 问：外界对价格相当好奇，请问能否透露明年产品推出时的价格细节？答：我们目前不向外公布这些信息。
　　4:05 问：谷歌将会如何将Chrome OS推向市场？媒体能否购买这些设备？答：我们也不知道。我们计划将数量有限的设备从现在起和明年1月份分发出去。
　　4:06 问：用户能否在Chrome OS上使用“本地”应用程序？你们是否将web定义为新的 本地应用？答：web应用就是本地程序。使用web技术，用户完全可以完成所有的任务 。
　　4:07 问：Web Store依赖Chrome浏览器的程度有多大，谷歌是否会向其他浏览器推出 这一应用商店？答：“Web Store的确依赖于Chrome的部分功能，但对于其中的部分应用，用户在任何浏览器中都可使用。皮采表示，安装应用导航界面和支付交易是Chrome原生的。
　　4:08 Chrome OS笔记本只能使用在Verizon网络下吗？答：用户不一定需要签约任何合 同。但CR-48系列是绑定Verizon的。
　　4:12 问：一台装有Windows XP的笔记本如果想要安装Chrome OS，可以如何操作？答 ：如果你懂得编写代码，你可以在任何设备上运行Chrome OS，但部分功能如验证启动功能需要新硬件支持，而这些硬件一般不会出现在标准PC上。
　　4:13 “我们希望为用户带来更加简单的使用体验，”用户不需要配置底层硬件。
　　4:14 问：这个系统看起来比Android更适合作为平板电脑操作系统。你同意吗？答：我们认为，Chrome OS和Android两种方式都非常令人兴奋……我随身同时携带Android 手机和Chrome OS笔记本。他们都是很重要的开源项目……部分OEM厂商将采用Chrome OS，但我们想让市场和用户选择他们喜爱的系统。两个系统都是谷歌的产品 ，因此我们都很乐意。
　　4:15 问：你们如何从Chrome Web Store中获得盈利？答：Chrome的改进带动了搜索量 的增长，直接影响了我们的营收。我们将提供企业应用。Web Store的目的是帮助用户找到应用。如果这个目标得以完成，我们会摸索出盈利模式的。
　　4:17 问：你刚才提到测试机型运行在英特尔凌动处理器平台，你们是否计划为 Chrome OS增加ARM平台支持？答：我们将Chrome OS设计成能够跨越多个平台的系统 。我们最早的合作伙伴计划使用英特尔架 构，因此我们将追随他们的步伐，支持合作伙伴。
　　4:18 问：CR-48的制造商是谁？答：CR-48只是参考硬件……它们不会上市销售…… 我们的合作伙伴将销售自己的设备。
　　4:19 问：我们什么时候能过见到数据在不 同的谷歌系统(Android、Chrome OS和 Google TV)之间繁殖？答：Chrome中的同步是一个云服务，我们将在合适的时间将其推广。
　　4:19 使用纽约时报应用相比直接访问纽约时报网站有什么优点？答：开发者们在利 用这个机会尝试不同的东西。有些开发者仅仅是想将他们的网站做成Chrome应用，有些则希望尝试不同的用户界面和不同的体验。在Chrome OS平台中，你可以使用GPS数据开发地理位置功能等。你可以将这些功能聚集到一个应用中。
　　4:24 问：用户能否在CR-48中接触到操作系统之外的东西？本地存储是如何进行的？ 答：Chrome OS有一个按钮，允许你对硬件进行任意操作，安装任何你喜欢的软件。
　　4:25 问：你们是否会推出谷歌品牌的Chrome OS笔记本，如果推出的话，你们是否会直接销售？答：不会，第一批上市的设备将来自三星。
　　4:27 谷歌表示，Chrome OS不一定非要绑定 到谷歌账户中。“你没有谷歌账户，也能使 用Chrome OS笔记本。”
　　4:28 问：Chrome OS是否有USB接口？答：我们正在对这一部分进行开发……我们已经能够支持所有的键盘和鼠标，不过尚不支持存储设备。对于打印设备，我们有云打印，我们认为那是更好的模式，因为用户不需要安装任何东西。
　　4:29 “今后是否会有其他的接口？当然会。 Chrome OS系统的配置会一直变化…… HDMI接口也在开发计划之列。”
　　4:29 “我们希望设备能够在不需要特定软件的情况下相互通信。”
　　4:30 问：你们是否会开发Chrome OS平板， 虚拟键盘是不是也很好？答：我们一直都 想开发一款拥有全尺寸键盘的产品，但我们希望其拥有触摸体验。我们目前没有具体的计划……我们将一步一个脚印地探索 。在先锋项目中，软件中存在一些bug。我们的目标是，先消除这些bug，然后再采取下一步的步骤。
　　4:31 问：你们是否计划在这些设备中支持4G或LTE连接？答：我们知道4G时代正在到来……我们正在同合作伙伴讨论。我们的目标是，确保为某一地区的领先运营商 提供选项。(张和)
