SharePoint 2010初览：在线编辑器大为改观
　　新浪科技讯  北京时间11月24日晚间消息，据国外媒体今日报道，微软最新发布的SharePoint 2010测试版相较SharePoint 2007在多处有了重大改进，包括改进的用户界面和在线编辑器功能。SharePoint 2010现在支持的浏览器数量更多，与微软Office办公软件也能更好地整合。
　　通过可视化升级功能，独立的附属站点能逐步从SharePoint 2007外观升级为SharePoint 2010外观。沙盒功能可在最小风险之下部署自定义代码。集成式的网络管理和富PowerShell脚本系统管理工具支持可简化并加速管理任务。Visual Studio 2010支持多种SharePoint 2010项目，并可检查已部署至SharePoint的代码是否有错误。
　　出于性能考虑，SharePoint 2010是一款64位软件，需要64位Windows Server 2008、64位SQL Server 2005或SQL Server 2008的支持。打算升级SharePoint的企业将需要把新硬件和基础设备的成本计入预算当中。目前微软尚未公布SharePoint 2010的定价。
　　一直以来，SharePoint都是一个适用于各种内部或公开网站的多功能平台，并专注于群体协作网站。SharePoint 2010在这些方面有了极大改进，变得更灵活和强大，拥有更佳的用户界面，也更适于部署多语种网站。尽管并非所有网站都要支持多种语言，但确有此需求的网站或许可以考虑用SharePoint 2010代替如Drupal这样的重量级开源网络框架。
　　无论从哪方面来看，SharePoint 2007的HTML编辑器都称不上强大，SharePoint 2010的在线编辑器则大为改观，可以说是目前最优秀的所见即所得(WYSIWYG)编辑器之一，就像是把微软Word的使用体验搬到了网络上。新的编辑器支持IE 7和IE 8以及火狐浏览器，在Safari浏览器中的效果也很出色。
　　SharePoint 2010对维基语法标识提供了近乎全面的支持，可以提供类似于维基百科的功能。该软件还提供了三种搜索模式，包括增强的企业级搜索引擎功能。此外还提供了SharePoint Workspace 2010桌面/笔记本电脑客户端，能提供离线同步功能。
　　用户既可以直接升级到SharePoint 2007，也可以在全新安装SharePoint 2010后使用SharePoint 2007的数据库备份，升级时数据库可以只读模式运行，以最大程度地减少宕机时间。此外比起SharePoint 2007并不完善的备份和还原工具，SharePoint 2010已大有改进。而且，开发者还可使用Visual Studio 2010开发工具，通过大量的API(应用程序接口)来增强SharePoint 2010的功能。(晓彬)
