Mac OS X Lion操作系统开售首日销量破百万
　　新浪科技讯 北京时间7月22日凌晨消息，苹果今天宣布，该公司最新推出的Mac OS X Lion操作系统开售首日销量已超100万份。
　　苹果去年10月在“Mac回归”大会上发布Mac OS X Lion，昨天正式开售。该公司今天在一份声明中称，Lion的销售速度“快过苹果曾经推出的所有操作系统”。
　　苹果全球产品营销高级副总裁菲利普·席勒(Philip Schiller)说：“Lion开局良好，用户评价很高，业界反应优良。这款操作系统是一个巨大的进步；它不仅新添创新功能，还使用户能够很方便地将Mac电脑升级至我们所曾推出的最优秀的操作系统。”
　　Mac OS X Lion是苹果首款通过Mac App Store提供数字版本下载的Mac操作系统。苹果计划下月开售U盘版“狮”，用户可以花费69美元购买一个存有新款操作系统的U盘，这一价格是数字版的29.99美元的两倍多。
　　目前，Mac OS X Lion在Mac App Store中获得9376次用户评分和2091个评论，平均评分为4.5分(满分5分)。
　　这是自2007年年底Mac OS X Puma发布以来，苹果首次公布Mac操作系统的销量。Puma发售2日销量超过200万份。(彦飞)
