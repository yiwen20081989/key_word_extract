谷歌Chrome Web Store网络商店将10月上线
　　【赛迪网讯】8月20日消息，据国外媒体报道，谷歌在本周早些时候证实，谷歌Chrome Web Store网络商店现在已经准备就绪。
　　据悉，一份报告表明，Chrome Web Store 网络商店将在10月上线。在19日，谷歌在博客中披露了关于网络商店进展情况的一些细节，但谷歌公司还没有透露具体的时间安排。
　　同时，谷歌确认，谷歌只收取5%的手续费，没有额外的收入份额。但是，登记成为Chrome开发商每次必须支付5美分。开发商可以免费、付费或者让用户免费试用应用软件。
　　Chrome Web Store网络商店可以提供3中类型的产品：应用软件、主题管理和应用扩展，开发人员可以对其中任何一项产品收费。谷歌需要足够数量的开发商，来使得其首款Chrome OS设备获得足够的重视。
