NVIDIA宣布携手微软共推高性能GPU计算
　　【赛迪网讯】9月29日美国消息，NVIDIA公司正式宣布，公司将与微软展开合作，借助Windows HPC Server 2008操作系统，共同推进NVIDIATesla图形处理器(GPU)在高性能并行计算方面的应用。
　　微软顶级计算副总裁Dan Reed表示：“GPU与CPU相得益彰，二者的有机结合为我们带来了多核协同处理的强劲性能以及巨大机遇。NVIDIA与微软所付出的汗水以及Windows HPC Server平台将帮助诸多领域的科学家与研究人员在各种应用程序上实现超级计算机的性能。”
　　NVIDIA研究院现已开发了多款面向Windows HPC Server 2008平台、基于GPU的应用程序，例如可用于超逼真汽车建模的光线追踪应用程序。与此相关的是，NVIDIA已经携手微软研究院打造了一个大型Tesla GPU计算集群，研究专为GPU而优化的应用程序。
　　此外，各种各样的企业级应用程序正在利用NVIDIA GPU的大规模并行CUDA架构来实现更高生产率。这些应用程序包括数据挖掘、机器学习、商业智能方面的应用程序以及分子动力学、金融计算和地震处理等科学应用程序。
　　CUDA架构让开发人员能够在一个协同处理模型内既使用CPU又使用GPU。应用程序的计算密集型部分利用GPU的并行计算动力，而应用程序代码的串行部分则在CPU上运行。
　　伊利诺伊大学厄本那香槟分校资深研究程序员John Stone表示：“GPU与Windows平台的珠联璧合对我们视觉分子动力学研究用户来说是巨大的福音，为数以千计的用户带来了高级的分子可视化以及强大的分析能力。随着研究工作朝着更大生物分子结构的方向发展，GPU将发挥越来越重要的作用。因为它们能够在这些可实现高度并行化的计算难题上展现出更加强劲的计算能力。”
　　NVIDIA公司Tesla事业部总经理Andy Keane表示：“科学界已经率先意识到，GPU能够彻底改变其工作面貌，在使用各种计算密集型应用程序时能够实现20至200倍的速度提升。由于Windows拥有Microsoft Visual Studio这样的强大开发工具、系统管理非常容易而且总拥有成本较低等原因，研究人员越来越多地在工作站以及数据中心中使用Windows操作系统。”
　　NVIDIATesla高性能GPU计算产品支持工作站的Windows XP与Windows Vista以及数据中心的Windows Server 2003和Windows Server 2008。
