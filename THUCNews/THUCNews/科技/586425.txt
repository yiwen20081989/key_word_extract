2008年微软十件大事：盖茨隐退居首
　　新浪科技讯 北京时间12月15日消息，据国外媒体报道，美国网站今天评选出了微软2008年十件大事，其中联合创始人比尔·盖茨(Bill Gates)隐退名列榜首，上榜的事件还包括收购雅虎风波、进军云计算领域等。
　　下面是2008年度微软十件大事简介：
　　1、盖茨隐退
　　今年微软管理层发生了较大变化，首当其冲的莫过于盖茨今年夏季退出公司日常管理工作，将更多时间投入到慈善事业中。
　　2、收购雅虎风波
　　2月1日，微软公布了以每股31美元、总价446亿美元收购雅虎的计划，遭到了雅虎拒绝。尽管微软后来将出价提高到了每股33美元，还是遭到了拒绝。微软5月份正式撤消了收购雅虎的方案。微软收购雅虎的目的是挑战谷歌在搜索及搜索广告市场上的霸主地位。
　　微软仍然公开宣称已经对收购雅虎不感兴趣，但由于雅虎11月宣布找到继任者后杨致远将辞去CEO一职，分析师认为微软、雅虎重启谈判的机率大大提高了。12月微软任命前雅虎搜索技术高管陆奇(Qu Li)出任网络业务部门掌门。
　　3、Vista有所改进，仍然恶评如潮
　　用户对Vista的不满之一是版本过多，使他们感到无所适从。由于存在大量软、硬件兼容问题，许多企业和消费者在购买预装Vista的系统后都将系统降级为Windows XP。就连微软的铁杆盟友英特尔也表示将推迟部署Vista。另外，微软还因“Vista Capable”营销活动遭到用户起诉，原告称微软的营销信息误导了他们。
　　Windows Vista的一个亮点是比XP更安全。
　　4、揭开Windows 7神秘面纱
　　微软5月份首次展示了Windows 7，演示了多点触控技术。Windows 7将使用与Vista相同的核心代码，支持相同的软件和硬件。但是，Windows 7的性能和对外设的支持将得到改进。在10月的“专业开发商大会”上，微软公布了明年初发布β版Windows 7的计划。
　　微软显然吸取了Vista的教训，一直在催促硬件厂商测试Windows 7与它们产品的兼容性。微软还宣布，Windows 7能够在上网本上运行。
　　5、Windows Server 2008获得好评
　　尽管Vista问题不断，但在用户和测评人员看来，Windows Server 2008的表现要好得多。它包含有Server Core版本，支持PowerShell命令行脚本。但美中不足的是，在发布之初，Windows Server 2008并没有集成虚拟软件。
　　微软还在利用Windows Server 2008开拓高性能计算市场。
