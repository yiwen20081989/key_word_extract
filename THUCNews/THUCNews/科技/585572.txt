微软十大产品树立其不败地位
　　[计世网消息](刘彦青 编译) 微软十大最棒产品:
　　动态服务器页面
　　尽管是个迟到者，但微软携动态服务器页面(ASP)技术强势进入了服务器端脚本领域。开发人员认为ASP不但简单易用而且足够灵活，表现非常优秀。
　　Exchange Server 2003
　　Exchange Server 2003的特征是管理简单、更好地支持第三方安全插件和服务器端ActiveSync支持。Exchange Server 2003用户很多，但几乎没有用户对它表示不满。Exchange Server 2003是微软的全能冠军选手。
　　Office 97
　　Office 97革命性的布局、工具条和语法工具非常引人注目。在此后10多年的时间里，Office 97的文件格式已经成为办公软件事实上的标准。
　　OneNote
　　OneNote是一款数字化笔记软件，个人用户和团队可以在任何笔记本电脑、台式机或平板PC上利用它获取、组织、发现、重用和共享笔记信息。OneNote兼具纸质笔记本的灵活性和数字媒体的功能，能够有效地提高用户的工作效率。
　　Visual Studio .NET
　　意识到Web将改变软件的开发方式后，微软决定修改其旗舰开发平台，满足新时代的需求。Visual Studio .NET采用以Web为中心的策略和新的编程理念，有助于维护微软与开发人员的密切关系。
　　Windows 2000
　　通过集成面向企业的Windows NT和面向消费者的Windows版本的特性(全32位支持和即插即用)，Windows 2000提高了系统稳定性，能够方便地进行配置。尽管随着时间的推移Windows 2000被发现存在大量漏洞，对于企业客户而言，Windows 2000是在服务器和客户端方面的一大进步。
　　Windows Powershell
　　尽管脱胎于DOS，但与Linux和Unix相比，Windows的命令行环境相当弱。通过Powershell，微软为Windows提供了一个强大的命令行环境，为Windows粉丝和开发人员提供了一种控制Windows系统的新方式。
　　Windows Server 2003
　　Windows 2000 Server引发了许多安全问题，微软意识到它必须作出重大改进才能够重新获得IT系统管理员的信任。与2000相比，Windows Server 2003采取了更严格的安全措施，大多数服务在缺省状态下都处于关闭状态。Windows Server 2003已经被证明是一款非常稳定和安全的服务器平台。
　　Windows服务器更新服务2.0(WSUS)
　　Windows服务器更新服务2.0启动了第三方补丁软件管理产业。由于微软在免费产品中提供了客户端补丁软件跟踪、政策控制和报告功能，第三方厂商被迫扩大了它们产品的功能。
　　《Halo》游戏
　　尽管严格说起来《Halo》并非微软的产品，但它是由微软发行的，也被看作是微软的产品。事实上，它可能是微软有史以来最成功和最棒的产品之一。《Halo》被认为极大地刺激了Xbox游戏机的销售量。
