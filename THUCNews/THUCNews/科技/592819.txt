微软停止为Windows Server 2000提供支持
　　新浪科技讯 北京时间7月14日凌晨消息，据国外媒体报道，微软周二称，该公司已经停止向Windows Server 2000服务器操作系统提供支持服务，并减少对Windows Server 2003操作系统的支持服务。
　　微软在其网站的帮助和支持页面上表示，对Windows Server 2000的所有公共支持都已在周二截止。Windows Server 2000用户将仅可使用自助网络支持服务，如FAQ和社区支持论坛等。此外，微软还已经停止对Windows Server 2003服务器操作系统提供主流支持服务，但用户将可在2015年以前继续接收安全性更新和获得付费支持服务。
　　不过，名为hotfix的特定补丁目前仅有购买了Extended Hotfix Support的用户才能使用。微软支持服务周期政策FAQ页面显示，该公司不会作出任何设计或特性更改。
　　市场研究公司eDigitalResearch称，从Windows Server 2003升级至Windows Server 2008等以后版本的成本将会很高。该公司技术部门负责人辛恩·怀特(Shane Wright)称：“这里所说的不是授权成本，而是转向另一种服务器配置的管理成本”，其中可能包括运行测试服务器、认证、回归、性能和兼容性测试等。(唐风)
