值得期待 韩国三阳发布3款手动定焦镜头
　　韩国三阳(Samyang)近日连续推出三款全手动镜头新品，它们分别是专门针对摄像领域的V-DSLR 8mm f3.5 Aspherical IF MC Fish-eye、改进型的AE 14 mm f/2.8 ED AS IF UMC以及拥有超大光圈的35mm f/1.4 IF UMC。目前，三阳公司尚未确定这些镜头的发售时间及价格(不过该品牌镜头一贯具有极高的性价比)。
　　V-DSLR 8mm F3.5 Aspherical IF MC Fish-eye：
　　此款8mm f/3.5鱼眼不同于之前的产品，它专为单反摄像服务，目前有佳能及尼康卡口版本。该镜头在光学结构上与前作没有区别，但前端口径更大，没有避光罩，而且镜身上的齿状对焦及光圈环非常适合安装额外的摄像用对焦、光圈调节装置。
　　光学结构方面，V-DSLR 8mm f3.5 Aspherical IF MC Fish-eye采用7组10枚镜片(内含1枚非球面镜片)，6枚光圈叶片，其最近对焦距离为30cm，重475g。
　　AE 14 mm f/2.8 ED AS IF UMC：
　　该镜头是前做的改良版本，不但镜组结构发生了变化，而且还追加了针对尼康单反的电子系统，如此一来在所有尼康单反上便可以实现自动测光功能(包括i-TTL)，而且还能完整记录EXIF信息。
　　光学结构方面，AE 14 mm f/2.8 ED AS IF UMC采用了10组14枚镜片结构，内含1枚非球面镜片及1枚复合型非球面镜片，它同样具有6枚光圈叶片，最近对焦距离为28cm，重530g。
　　35mm f/1.4 IF UMC：
　　此款镜头有望成为一支性价比极高的多用途镜头，在全画幅单反上市完美的人文利器，在APS-C单反上变身为标准镜头，而在4/3系统上则可用于人像拍摄。
　　在光学结构上，35mm f/1.4 IF UMC采用10组12枚镜片结构，内含1枚非球面镜片和2枚高折射率镜片，其光圈叶片为6枚，最近对焦距离为30cm，滤光镜口径77mm。
