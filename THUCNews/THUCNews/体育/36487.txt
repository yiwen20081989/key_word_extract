英超半程最佳：利物浦已难寻踪迹 华人主席成最大黑马
　　门将 < 哈特(19场失18球)
　　相比于四强门将屡屡失手，被曼城租借到伯明翰的哈特让人眼前一亮。如果哈特能继续这种神勇表现的话，他无疑将会成为今年夏天三狮军团的一号门将。
　　右后卫 < 伊万诺维奇(14场/0球)
　　如果以是否被皇马觊觎作为一流球员标准的话，伊万诺维奇现在俨然已跻身顶级球星行列。在博辛瓦伤病缠身的情况下，可以同时胜任中路和右后卫的塞尔维亚人为蓝军12月前的钢铁防线发挥了巨大作用，他为德罗巴送出的精准传中也令人印象深刻。
　　中卫 < 邓恩(19场/3球)
　　现在赋闲在家的休斯可能还在为夏天放走邓恩而后悔，曼城的后防在邓恩走后立刻显得散乱不堪，而这位爱尔兰老将也在维拉焕发出第二春，在对阵切尔西和曼城时的关键进球更是价值千金。
　　中卫 < 费尔马伦(19场/4球)
　　由于上半赛季的出色表现，费尔马伦被很多媒体推为本赛季英超的最佳引援，他用行动回击了当年亲赴阿贾克斯考察的阿森纳传奇中后卫亚当斯的“身高不足，身体条件不适合英超对抗”论断。除了在后防线上与加拉配合日渐默契之外，脚法出众、嗅觉敏锐的费尔马伦还是球队的得分手之一。
　　左后卫 < 埃夫拉(20场/0球)
　　尽管本赛季曼联后防整体不佳，但埃夫拉却是唯一一位联赛全勤且状态没有下滑的红魔后卫。虽然阿什利·科尔同样出色，但考虑到埃夫拉独撑危局的难度和勇气，法国后卫的半程最佳实至名归。
　　右前卫 < 伦农(18场/3球)
　　“我唯一的布置就是让球员尽量把球交给伦农，直到他突破突到恶心为止。”主场3球横扫曼城一役，热刺主帅雷德克纳普对伦农如此评价。凭借着充沛的体能，令人匪夷所思的速度，已为球队献上9次助攻的伦农无疑是英超上半段表现最突出的边路选手。
　　后腰 < 埃辛(14场/3球)
　　虽然最近在病榻上休养了大半个月，但仍然没有哪个防守中场能在本赛季达到埃辛的高度。加纳人是蓝军钻石中场中适应速度最快的球员，除了能为后防提供屏障般的保护，他招牌式的远射也堪称一绝。他缺阵后，蓝军的成绩下滑也直接凸显出了他的价值。
　　前腰 < 法布雷加斯(16场/9球)
　　9粒进球，12次助攻，阿森纳的433新阵彻底释放出了小法的能量，本赛季进一步成熟的西班牙人无疑是英超现在表现最为惹眼的中场进攻球员。担任枪手队长一年有余，如今的小法更多了几分统治力，对热刺的闪电一击和替补上场27分钟降服维拉的表现让人们开始将他奉为“征服者塞斯克”。
　　左前卫 < 贝拉米(16场/6球)
　　前锋出身的贝拉米是本赛季曼城的亮点之一，他在左路的高速突破让对手猝不及防，阿森纳和曼联都先后在威尔士人犀利的反击下大乱阵脚。尽管休斯离去，但贝拉米的实力还是得到了新帅曼奇尼的认可，在曼奇尼任上夺取联赛冠军，是贝拉米的目标。
　　右前锋 < 德罗巴(18场/14球)
　　虽然领跑射手榜的球员多达3人，但其中的最佳无疑是德罗巴。要想知道对手对德罗巴有多忌惮，从世界杯抽签仪式上成功避开科特迪瓦后英国媒体如释重负的心态中就可窥见一斑。本赛季除了进球如麻之外，魔兽还送出8次助攻，团队精神也让正处于巅峰的德罗巴更可怕。
　　左前锋 < 鲁尼(19场/14球)
　　和同样攻入14球的迪福相比，鲁尼在球队中的作用更为关键，在曼联进攻思路还并未理顺的情况下，他是红魔的绝对主心骨，14个联赛入球中有8粒是首开纪录，已戴上国家队队长袖标的小胖正迈向自己的巅峰。
　　最佳教练 < 霍奇森
　　和手下强将如云的四强名帅以及习惯性弃欧战主攻联赛的奥尼尔相比，这位喜欢在天空电视台演播室蹭球看的本土老帅用一套单薄的阵容成功应对了双线作战的压力：欧联杯小组成功突围；联赛成绩稳定，让人们相信上赛季联赛第七的富勒姆绝非是昙花一现。如今，克拉文农场已成为豪门谈之色变的险恶之地，而霍奇森也被热捧为卡佩罗合同期满后的继任。
　　最佳主席 < 杨家诚
　　当这个略显神秘的香港人最终完成伯明翰的收购的时候，没有人会想到当时还在为保级挣扎的伯明翰会青云直上，甚至一度跻身前六，连续11场不败7胜4平的战绩让豪门也汗颜不已，今年的冬季转会市场将是进一步考验这位华人老板魄力的舞台。
