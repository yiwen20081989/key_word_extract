特维斯曼联欧冠5对手无国米 红魔不留我想去皇马
　　
　　
我感谢弗格森
　　欧洲俱乐部最高荣誉欧洲冠军杯，南美俱乐部最高荣誉解放者杯，世界俱乐部最高荣誉世俱杯?2月5日刚刚度过25岁生日的特维斯，已经成为第一位拥有上述三项荣誉的阿根廷人。本赛季，特维斯还要与曼联一起冲击历史上从未有球队完成过的4冠伟业：欧冠、英超、足总杯、联赛杯。或许，对特维斯而言，这可能也将是他在曼联的最后一个赛季。?
　　本赛季结束后，曼联与特维斯的两年租借合同即将到期。按照曼联与特维斯所有者——MSI投资集团的协定，曼联可以在2009年1月31日前以3700万欧元的价格买断特维斯。但由于受到经济危机影响，曼联并没有在上述日期前行使对特维斯的优先购买权，特维斯赛季结束后进入转会市场，已成必然。
　　特维斯的讲话在很多人看来总是缺乏必要的修饰，但对采访者而言，你会喜欢和特维斯坐在一起。
　　谁不喜欢为皇马踢球？皇马是世界上最伟大的俱乐部之一，但能否转会不取决于我。
　　DB 越来越多的人开始谈论你的未来，比如赛季结束后离开曼联甚至英超，你是怎么想的？
　　CT 事实上，我现在很喜欢曼联和英超，在这里踢了三个赛季后，我已经完全适应了。我现在只希望自己的出场时间可以再多一些。
　　DB 去年夏天贝尔巴托夫加盟曼联后，你的出场时间就开始减少。所以在很多人看来，这个夏天你很可能会加盟皇马，你个人怎么看？
　　CT 谁不喜欢为皇马踢球？与曼联一样，皇马是世界上最伟大的俱乐部之一，那里还有很多我认识的阿根廷球员?但能否转会不取决于我。
　　DB 如果加盟皇马，你就可以与梅西在世纪大战中较量了。
　　CT 梅西是我的朋友，我们已经在上赛季的冠军联赛中交过手了。在我看来，他将成为2009年世界最佳球员。
　　DB 你的队友C。罗纳尔多已经夺得了2008年所有个人荣誉?
　　CT 是的，因为在2007-08赛季，克里斯蒂亚诺与曼联夺得了英超和欧冠。但我想，梅西也会在2009年获得一切个人荣誉。
　　DB 你为什么可以得到很多球迷的喜爱？
　　CT 我想是因为我从来没有任何的隐藏和保留，我总是为我身穿的球衣奉献一切，球迷可以看到这些。比如在西汉姆联，我们并没有获得任何荣誉，但我们用一种令人难忘的方式保级成功?球迷知道我的心。
　　DB 你是否知道，你是第一位夺得解放者杯、冠军杯和世俱杯的阿根廷球员？
　　CT 是的，已经有人告诉我了。这太难以置信了，我从未想过自己在曼联的第一年，就可以夺得英超和欧冠，这也让我们对这个赛季有了更高的要求。
　　DB 很少有球员可以第一次参加欧洲冠军联赛就捧杯。
　　CT 是的，这是我职业生涯发生过的最美妙的事情之一。
　　DB 上赛季，你总共参加了39场正式比赛，打入19球；更让人意外的是，你还奉献了18次助攻?
　　CT 作为射手，我当然喜欢进球得分，但对我来说球队的胜利永远是最重要的。如果有队友比我的位置更好，我更喜欢传球，而不是自己射门。
　　DB 本赛季曼联是否可以蝉联英超和欧洲冠军杯？
　　CT 我想可以。就像博卡、河床或者AC米兰一样，世界上任何伟大俱乐部对荣誉都是永远充满饥饿感，曼联也不例外。我们希望可以创造历史。
　　DB 那曼联卫冕欧冠的主要对手都有哪些？
　　CT 我认为皇马肯定是其中之一。此外，还有巴塞罗那、拜仁、阿森纳、切尔西?尤其是巴萨，他们是本赛季表现最出色的球队之一。
　　曼联所有人都叫他“亚历克斯”，而不是爵士或者弗格森爵士，偶尔也会叫他“老板”。
　　DB 本赛季是你在曼联的第二个赛季，你是否已经适应了球队和这座城市？
　　CT 是的，我的感觉非常好，我认为来到这里的决定是非常正确的。在曼联，锋线的竞争总是非常困难，但亚历克斯一直肯给我机会，我很感谢他。
　　DB 弗格森允许你直接叫他的名字亚历克斯？
　　CT 当然。球队所有人都叫他“亚历克斯”，而不是爵士或者弗格森爵士，偶尔也会叫他“老板”。弗格森看上去是一位非常严肃的教练，但事实恰恰相反，他很随和，非常喜欢和球员交流。我刚来到曼联还没有完全适应球队的时候，总是受到媒体批评，是他站在我身后，告诉我要耐心冷静——这样的教练可以让球员感受到信任。
　　DB 作为教练，弗格森怎么样？
　　CT 最让我感到惊讶的，是我来到曼联以后才知道，他从来不指挥球队训练，一切训练都是由奎罗斯(前助理教练)带队完成，他只是坐在教练席或者看台上观看。但他在比赛中的指挥却是天才级别的。
　　DB 弗格森是否改变了你踢球的方式？
　　CT 不，他给了我很多自由。他对我唯一的要求，就是要多帮助中场球员防守和尽量多在前场阻截，减缓对方的进攻。这对我来说也不是改变，因为在博卡、科林蒂安和阿根廷国家队，我一直是这么踢球的。真正让我改变踢球方式的，是英超的比赛风格。在阿根廷或者巴西联赛，球员有很大的空间，你可以更长时间地控球；但英国足球更强调身体，跑动很多，空间却很小。你必须快速判断和快速带球、传球，否则就会挨踢。
　　DB 你的职业生涯遇到过两位传奇教练：弗格森和比安奇。为什么他们都可以取得成功？他们有什么共同点吗？
　　CT 有，他们都善于和球员沟通，都有对胜利的强烈渴望。
　　DB 谁是你在曼联的朋友？
　　CT 我在曼联已经有很多好朋友了，比如里奥(费迪南德)、克里斯蒂亚诺(罗纳尔多)和鲁尼，跟我关系最好的是埃夫拉。但这里和阿根廷不同：在英格兰，友谊更多是表现在训练场和球场上，训练结束或比赛结束后，球员就会回到自己家；而在阿根廷，训练结束后大家会聚在一起聊天、吃饭，甚至还经常会约定某一天集体去做些什么事情。
　　DB 在很多人眼中，鲁尼永远是个长不大的孩子，麻烦不断?
　　CT 麻烦不断？我想是对手才会这样认为吧。因为只要给他半米空间，鲁尼就可以攻破对手的球门。在我看来，鲁尼是一位非常聪明的球员，总知道如何摆脱对手的盯防。
　　DB C罗呢？
　　CT 和梅西一样，C罗是目前世界上最出色的球员之一。他本赛季能继续留在曼联，让俱乐部所有人都感到快乐。
　　只要国家队需要，我会踢任何位置；即使不让我上场，我也会在场外为队友加油。在我看来，能够进入国家队，已经是一种至高无上的荣誉。
　　DB 现在阿根廷国家队主教练成了马拉多纳，这一变动对你来说意味着什么？
　　CT 对我来说就像美梦成真，因为所有人都知道迭戈对于阿根廷、国家队和我们这一代球员意味着什么。
　　DB 2008年11月19日，马拉多纳执教国家队首战击败苏格兰后，你是与他拥抱时间最长的球员之一，这有什么特殊含义吗？
　　CT 有，因为在比赛之前，我和他曾经进行过一次只有我们两个人才知道内容的谈话。
　　DB 目前阿根廷队的锋线有很多选择：拉韦齐、萨拉特、迭戈·米利托、阿圭罗，还有梅西。你是否担心自己会失去位置？
　　CT 只要国家队需要，我会踢任何位置；即使不让我上场，我也会在场外为队友加油。因为在我看来，能够进入国家队，已经是一种至高无上的荣誉。
　　DB 马拉多纳将国家队队长袖标交给了马斯切拉诺，你曾经和他一起效力过科林蒂安和西汉姆联，你如何评价这位新队长？
　　CT 任何主教练都有选择队长的权力。在迭戈眼中，马斯切拉诺所在的位置是球队中最重要的，我认为这很正常。
　　DB 你和马斯切拉诺的关系怎么样？
　　CT 我们不是朋友，平时很少开玩笑，但关系还不错，因为我们相互尊重对方。
　　DB 当曼联遇到利物浦，你们会在比赛前说些什么吗？
　　CT 什么都不会说。这听起来有些不可思议，但我们的确只会相互问候对方，不会做什么交流。我跟马斯切拉诺在科林蒂安和西汉姆联当过队友，但我们不是朋友，平时也很少开玩笑。
　　DB 2009年11月，阿根廷将在马德里与西班牙国家队进行一场友谊赛，这也是目前世界上公认状态最好的两支球队的对决，你怎么看待这场比赛？
　　CT 我想这将是一场非常精彩的比赛，西班牙拥有很多出色的球员，是阿根廷在2010年世界杯上的主要对手之一。
　　DB 在你看来，哪些球队将成为南非世界杯的夺冠热门？
　　CT 我想意大利队仍然是热门，此外我也很喜欢葡萄牙队。当然，英格兰、巴西、阿根廷都是冠军的争夺者。夺得2010年南非世界杯是我的梦想，我想阿根廷队有实力做到这一点，因为几乎没有哪支球队可以同时拥有里克尔梅、梅西、马斯切拉诺和加戈这样不同位置、却都是顶级水平的球员。
　　DB 你是否会在博卡青年结束职业生涯？
　　CT 我希望如此。
　　DB 最后问一句，你胳膊上的文身代表着什么？
　　CT 我将它献给我的叔叔特罗帕。因为过去几年，他一直在和癌症作斗争，他是个真正的斗士。 (编译/吴玉)
